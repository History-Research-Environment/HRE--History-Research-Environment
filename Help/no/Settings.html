HRE-innstillinger

Denne skjermen lar deg angi en rekke preferanser for hvordan HRE fungerer. Innstillingene kan endres når som helst og påvirkes ikke av lukking og gjenåpning av HRE eller dine prosjekter.

1. Velg INNSTILLINGER fra hovedmenyen VERKTØY, deretter BRUKER for å velge og angi en rekke brukeralternativer.

2. Fra innstillingssettet med faner kan brukeren velge å endre:

HRE Utseende: for å sette 'utseendet' - tema / font størrelse og type (den HRE standard er Graphite, Blue-medium-font, Arial)

Språkinnstillinger: for å angi standardspråk for HRE-skjermer, data og rapporter

Formatinnstillinger: for å stille inn dato og klokkeslett formater

Brukeralternativer: for å velge visning av velkomstskjermen, Avbryt meldinger, Åpning av det siste prosjektet lagret på HRE re-start, om å Backup et prosjekt som den lukkes og om HRE Plugins bør være aktivert

Accents: for å sette fargene på skrift og bakgrunn for eventuelle HRE Temaer du ønsker å definere

Filplassering: for å sette standardplasseringene for HRE prosjekter, rapporter, sikkerhetskopiering og eksterne sikkerhetskopiering av filer

Feilsøk alternativer: for å sette skriving av en loggfil (standard), skriving av debug meldinger og timing meldinger. Disse alternativene er primært for bruk av HRE teknisk støtte

Server Modus: for å muliggjøre gjennomføring av HRE i server-modus (standard er AV)
