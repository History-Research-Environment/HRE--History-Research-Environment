Accesso al database HRE

Questa finestra di dialogo viene visualizzata quando tenti di aprire un progetto HRE definito su un server remoto.

Lo schermo visualizza 3 aree di immissione dati, come segue:
1. Dati di connessione remota, costituiti dalla tipologia Database; indirizzo IP del server; numero di porta del server. Queste informazioni servono principalmente a scopo di conferma e normalmente non ci si aspetta che tu le modifichi

2. Il tuo ID di accesso per questo progetto, fornito dall'amministratore di sistema del server remoto e salvato da HRE

3. La tua password per questo progetto, come inizialmente fornita dall'amministratore di sistema del server remoto

NOTA:

* HRE consentirà solo 3 tentativi di accesso prima di fallire il processo di accesso.

* L'utilizzo di un server remoto richiede che HRE sia in "modalità server": questa impostazione è impostata in Strumenti->Impostazioni->Modalità server.
