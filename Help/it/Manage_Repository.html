Gestisci Repository

La schermata Gestisci Repository consente di visualizzare, modificare, copiare, aggiungere ed eliminare i dati del Repository.

L'utilizzo delle funzioni 'Aggiungi' e 'Modifica' visualizza la schermata dei dettagli del Repository, che consente di impostare il nome del Repository, il nome abbreviato, la posizione (in base allo stile di posizione selezionato) e un promemoria.

Un Repository in uso non può essere eliminato.

I Repository possono essere trovati nell'elenco inserendo del testo nella casella di testo 'Trova'.
