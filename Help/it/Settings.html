Impostazioni HRE

Questa schermata consente di impostare una serie di preferenze per il funzionamento di HRE. Le impostazioni possono essere modificate in qualsiasi momento e non sono interessate dalla chiusura e dalla riapertura di HRE o dei tuoi Progetti.

1. Dal menu principale STRUMENTI, seleziona IMPOSTAZIONI, quindi UTENTE per selezionare e impostare una gamma di opzioni Utente.

2. Dall'insieme di schede puoi scegliere di modificare:

    Look and Feel HRE: Per impostare il "aspetto e la sensazione" - il tema / dimensione del carattere e digitare (il valore predefinito HRE è Graphite, Blue-medium-font, Arial)

    Impostazioni della lingua: Per impostare i linguaggi predefiniti per le schermate HRE, i dati e i report

    Impostazioni formato: per impostare i formati di data e ora

    Opzioni utente:

  • Per visualizzare la schermata di benvenuto all'avvio di HRE (l'impostazione predefinita è No)
  • Per mostrare le richieste di conferma per Annulla e altre azioni (l'impostazione predefinita è Sì)
  • Per aprire l'ultimo progetto HRE chiuso all'avvio di HRE (l'impostazione predefinita è No)
  • Per eseguire il backup di un progetto quando viene chiuso (l'impostazione predefinita è No)
  • Per abilitare l'elaborazione del codice "plugin" (l'impostazione predefinita è No)
  • Per ricaricare sempre il Selettore Persona quando una persona viene aggiunta al database (l'impostazione predefinita è Sì). Se questa opzione è disabilitata, il Selettore Persona viene ricaricato solo quando lo si richiede dal menu HRE
  • Per richiedere sempre il nome del coniuge quando si salvano i matrimoni (l'impostazione predefinita è No)

    Accenti: Per impostare i colori del carattere e dello sfondo per tutti i temi HRE che desideri definire

    Posizioni dei file: Per impostare le posizioni predefinite per progetti di hre, report, backup e backup dei file esterni

    Opzioni di debug: Per impostare la scrittura di un file di registro (predefinito), la scrittura di messaggi di debug e messaggi di temporizzazione. Queste opzioni sono principalmente per l'uso da HRE Supporto tecnico

    Modalità server: per abilitare l'esecuzione di HRE in modalità server (predefinito è No)
