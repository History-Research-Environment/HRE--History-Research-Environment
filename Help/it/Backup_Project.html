Backup di un progetto HRE

Questa schermata consente di eseguire il backup di un progetto HRE esistente, indipendentemente dal fatto che il progetto sia attualmente aperto in HRE o meno. È possibile selezionare la destinazione del file di backup e scegliere se eseguire o meno il backup anche delle cartelle esterne che fanno parte del progetto. La posizione di backup predefinita può essere impostata in Strumenti > Impostazioni > Utente > Posizioni file.

1. Selezionare PROGETTO >> PROGETTO DI BACKUP ed evidenziare il Progetto desiderato dall'elenco presentato

2. HRE visualizzerà ora il nome del file che verrà applicato al backup, che è il nome del progetto con l'ora e la data correnti aggiunte. A questo punto c'è un'opzione per visualizzare un RIEPILOGO del Progetto selezionato

3. Fare clic su SFOGLIA per individuare la cartella in cui memorizzare il backup. Sebbene venga visualizzata la cartella predefinita (come definito nelle Impostazioni HRE per le posizioni dei file), è possibile navigare per selezionare qualsiasi altra posizione. Al file verrà assegnato automaticamente il suffisso del file di backup compresso di .hrez. Una volta che la cartella della posizione desiderata è nella finestra, premere INVIO per selezionarla

4. Ora puoi scegliere di eseguire il backup anche delle cartelle di file esterne (immagini, documenti, file audio, video, ecc.) associate a questo progetto spuntando la casella di selezione. Si aprirà una nuova finestra che ti consentirà di selezionare una o più cartelle di cui eseguire il backup e di specificare dove vuoi salvarle, che potrebbe essere diversa dalla posizione di backup del progetto. Il nome del file di backup conterrà il nome della cartella originale e sarà un tipo di file zip standard.

5. Ora fai clic su BACKUP per avviare il processo. Se il progetto nominato è attualmente aperto in HRE, verrà prima chiuso e quindi inizierà il backup. Verrà eseguito per primo il backup del database del progetto, seguito dalle cartelle di file esterne (se specificate).

6. In alternativa, seleziona ANNULLA se il backup non è più necessario.
