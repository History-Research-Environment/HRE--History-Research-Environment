Elimina un progetto HRE

Questa schermata consente all'utente di eliminare un progetto in eccesso.

Nota che il progetto non può essere attualmente aperto in HRE: deve prima essere chiuso.

1. Seleziona il progetto che desideri eliminare. Il nome del progetto, il nome del file del progetto e la posizione verranno quindi inseriti sotto la tabella di selezione

2. A dire il vero, in questo momento potrebbe essere saggio controllare il SOMMARIO del progetto

3. Se procedi, premi CANC.

4. Mantieni database: ti verrà chiesto se desideri conservare il file del database (durante l'eliminazione del riferimento al progetto HRE) o eliminare entrambi. Rispondi NO per eliminare tutto o SÌ per conservare il file del database.
