COPIA DI UN PROGETTO HRE

Questa schermata consente di copiare un progetto HRE da qualsiasi sorgente a qualsiasi destinazione scelta. Può quindi essere aperto come richiesto. La posizione di copia predefinita può essere impostata in Strumenti > Impostazioni > Utente > Posizioni file.

1. È possibile selezionare un progetto dall'elenco dei progetti noti o utilizzare BROWSE per aprire la finestra di selezione file per trovare un file HRE su questo computer, su un'origine remota o su un'unità rimovibile. Seleziona il progetto richiesto

2. In COPY TO dai un nome al nuovo progetto

3. Usa nuovamente Selezione file per SFOGLIA per la cartella di destinazione del progetto e assegna il nuovo nome file

4. Premi COPIA per avviare il processo

5. Questo progetto ora può essere aperto su questo computer o su un altro computer.
