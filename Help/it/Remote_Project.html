DEFINIZIONE DEL PROGETTO REMOTO

La finestra di dialogo Definizione progetto remoto consente di definire un progetto HRE esistente su un server remoto o di modificare tale definizione, in base alla selezione di una di queste due funzioni nella schermata iniziale.

La selezione di "Modifica un progetto remoto" consente di scegliere un progetto remoto dall'elenco di tutti i progetti attualmente noti alla propria installazione HRE.
Una volta selezionato un progetto remoto, la schermata si espande per mostrare 2 sezioni di informazioni sul progetto per la modifica.
Scegliendo "Definisci un nuovo progetto remoto" vengono visualizzate le stesse 2 sezioni di informazioni sul progetto, in questo caso per l'immissione iniziale.

Il primo set di informazioni è la definizione del progetto remoto: nome, nome file e posizione sul server remoto.
La seconda serie di informazioni definisce il server remoto: il suo nome, indirizzo IP e numero di porta per l'accesso.

Tutte queste informazioni ti verranno fornite dall'amministratore del server remoto, che avrà creato il progetto HRE a cui desideri accedere.

Una volta completate le informazioni, è possibile salvarle. L'accesso al progetto HRE remoto definito sarà quindi possibile utilizzando la normale schermata HRE "Apri progetto".
