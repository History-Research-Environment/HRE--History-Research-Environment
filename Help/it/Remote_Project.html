Definizione Progetto Remoto

La finestra di dialogo Definizione Progetto Remoto consente di definire un Progetto HRE esistente su un server remoto o di modificare tale definizione, in base alla selezione di una di queste funzioni nella schermata iniziale.

Selezionando 'Modifica un Progetto Remoto' è possibile scegliere un progetto remoto dall'elenco di tutti i progetti di questo tipo attualmente noti all'installazione HRE.
Una volta selezionato un progetto remoto, la schermata si espande per mostrare 2 sezioni di informazioni sul progetto da modificare.
Scegliendo 'Definisci un Nuovo Progetto Remoto' vengono visualizzate le stesse 2 sezioni di informazioni sul progetto, in questo caso per l'inserimento iniziale.

Il primo set di informazioni è la definizione del Progetto Remoto: il suo Nome, Nome File e Posizione sul server remoto.
Il secondo set di informazioni definisce Server remoto: nome, indirizzo IP e numero di porta per l'accesso.

Tutte queste informazioni ti saranno fornite dall'amministratore del server remoto, che avrà creato il progetto HRE a cui desideri accedere.

Una volta completate le informazioni, potrai salvarle e accedere al progetto HRE remoto definito tramite la normale schermata "Apri progetto" di HRE.
