Ripristina un progetto HRE di cui è stato eseguito il backup

Questa schermata consente di ripristinare i backup precedenti e quindi di lavorare con quel file come progetto aperto corrente.
Il Progetto originale, se ancora presente, verrà sovrascritto solo se l'utente consente espressamente che ciò avvenga.

1. Fare clic su TROVA UN FILE DI BACKUP e dal Selettore file selezionare il file che si desidera ripristinare; in alternativa, vai in qualsiasi altra posizione e SELEZIONA il file

2. La tua selezione, incluso il suo nome descrittivo, sarà ritrasferita alla finestra Project Restore. Controlla le NOTE sulle opzioni da qui

3. Imposta la posizione della cartella in cui ripristinare il backup, sempre dalla finestra Selezione file

4. Il nome file predefinito è quello utilizzato dal progetto originale, ma DEVE essere modificato se si intende utilizzare la stessa destinazione in modo da non sovrascrivere il file originale

5. È fondamentale che al progetto restaurato venga dato un nuovo nome descrittivo per differenziarlo dal progetto originario. Una volta inserito questo, fare clic su RIPRISTINA

6. Una volta completato, il progetto ripristinato sarà il progetto attivo

7. Dopo aver verificato, se il progetto ripristinato è ciò che era necessario, in seguito puoi scegliere di eliminare la vecchia versione del progetto se è ancora su questo computer.

8. La seconda parte della schermata consente il ripristino di file esterni (immagini, documenti, ecc.) dai file zip di backup creati in Project Backup. Nota che le due parti dello schermo funzionano indipendentemente, in altre parole, puoi ripristinare un file esterno senza prima ripristinare un file di progetto.
