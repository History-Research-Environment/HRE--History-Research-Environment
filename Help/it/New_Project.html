CREA UN NUOVO PROGETTO

Questa schermata consente di creare un nuovo progetto, definirne il nome e il nome file e designare la posizione in cui archiviare i file del database.

Nota che questo NON deve essere utilizzato per importare un set di dati TMG. Il processo per Importazione di dati da TMG viene selezionato separatamente dal menu STRUMENTI.

Selezionare Crea progetto dal menu Progetto fornisce un processo in tre fasi da seguire:

1. Assegna un nome al tuo progetto per identificarlo in futuro. Dovrebbe essere descrittivo e può essere composto da più parole, ma deve essere unico

2. Fare clic su BROWSE per scegliere dove memorizzare il database. Verrà presentata una posizione predefinita, ma puoi sceglierne un'altra se necessario

3. Una volta selezionata la posizione del file, assegna al tuo progetto un nome file conciso, che deve essere diverso da qualsiasi altro file di progetto HRE archiviato in quella posizione. È necessario seguire le convenzioni Windows/MAC relative ai caratteri che è possibile utilizzare nei nomi dei file e HRE aggiungerà automaticamente il suffisso del file .mv.db. Premi SELEZIONA.

Ora la finestra sarà completa mostrando il tuo nuovo nome progetto, nome file e posizione

A questo punto puoi scegliere di ANNULLARE o CREA il nuovo Progetto.

Se ora hai creato il progetto, verrà inizializzato, aperto e reso ATTIVO. Apparirà una finestra per indicare che il processo è stato completato.

Ora puoi iniziare a inserire i dati nel nuovo progetto o scegliere di passare a un altro progetto aperto in modo che sia attivo o aprire un altro progetto.
