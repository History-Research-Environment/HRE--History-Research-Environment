ICONE DELLO SCHERMO HRE

Queste icone forniscono un accesso rapido ad alcune funzionalità e strutture di HRE.

Aiuto: fare clic per visualizzare le pagine della Guida appropriate per lo schermo

Output: fare clic per produrre qualsiasi tabella sullo schermo in una stampa o in un file csv/tsv

Promemoria: fai clic per aprire la finestra di dialogo Promemoria, in cui puoi creare note di testo sull'utilizzo dello schermo

Configura: fare clic per configurare le tabelle visualizzate sullo schermo

Copia persona: Fare clic per copiare la persona attualmente visualizzata

Elimina persona: Fare clic per eliminare la persona attualmente visualizzata

Rinumerare la persona: Fare clic per rinumerare la persona attualmente visualizzata
