Rapporto di registrazione HRE

Questa schermata consente all'utente di esaminare i file di registro delle attività di HRE creati ogni volta che viene aperto HRE e di scegliere di eseguire diverse attività su tali file.

1. Seleziona RAPPORTI >> REGISTRAZIONE

2. Fare clic su BROWSE per aprire la cartella predefinita per i file di registro HRE, come mostrato

3. È ora visualizzato un elenco dei file di registro giornalieri e facendo doppio clic su uno di questi si aprirà il file, che mostra tutta l'attività in HRE durante quel particolare giorno

4. Il file di registro può contenere una grande quantità di dati sotto le intestazioni: Data, Ora, Server (Computer), Nome progetto e Voce di registro. Nota che quest'ultima colonna può includere azioni del programma così come errori

5. È ora possibile emettere il contenuto del registro, o una parte selezionata di esso, su un file o una stampante. Questo può essere utile per il supporto tecnico o per ripercorrere i passaggi effettuati in una sessione HRE. Per Emettere tutto o una Selezione usa il mouse per evidenziare le righe del registro che hai scelto. La destinazione può essere un file o una stampante, incluso un file PDF. Una volta che è stato fatto, facendo clic su Reimposta selezione si ripristina l'intero file di registro per visualizzarlo ancora una volta o consente la scelta di un altro file di registro.

6. Al termine, chiudi la finestra di registrazione.
