HRE SCREEN ICONS

These icons provide quick access to some of the HRE features and facilities.

Help: Click to display appropriate Help pages for the screen

Output: Click to output any screen table to either printout or a csv/tsv file

Reminder: Click to open the Reminder dialog, where you can create text notes on use of the screen

Confgure: Click to configure tables shown on the screen

Copy Person: Click to copy the currently displayed person

Delete Person: Click to delete the currently displayed person

Renumber Person: Click to renumber the currently displayed person
