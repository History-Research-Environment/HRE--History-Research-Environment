HELP (Main Menu)

From HELP >> Help Contents, you can carry out several tasks in the Help Navigator, including browsing or searching Help topics, and printing out a copy of any chosen topic.

1. The Help Contents screen displays the Help system's Table of Contents and from it you can choose topics to view, view the topic Index or Search for certain items

2. Each entry with a book symbol indicates there is an expanded list of items which can be revealed by double-clicking the book, or clicking once on the plus + sign.
Double-click the open book or single click the minus - box to close the topic

3. Double-clicking on individual items opens the 'topic window' for that particular item.

The MAIN MENU AND TOPIC LIST

1. The FILE menu provides options to:
- DISPLAY an open topic (also viewed by double-clicking the topic)
- DISPLAY another topic in a new window (more than one topic Open, with windows stacked)
- PRINT the Help tree structure
- PRINT the topics (a large booklet!)
- CLOSE the main window so as to just view an open topic file
- EXIT from the HELP Contents completely, including open topic windows

2. The VIEW menu provides the option to view the tabs: Contens, Index or Search

3. The HELP menu shows current version details of the Help software.

CONTENTS TAB

Topic Screens

1. On each topic screen, you can read through the Help file

2. The FILE menu allows you to Print the topic, Close that topic window or Exit from the Help files

3. The GO menu allows you to go back or forward through topics you have browsed in this session, and this can also be accomplished via the hot keys (ALT-left or ALT-right) or by using the arrow icons

4. The TOOLS menu allows you to go back to the Help Navigator window (if you had earlier closed it) to search other topics, Copy highlighted text (CTRL-C), Highlight all text (CTRL-A), Find words or phrases in that topic and increase or decrease the topic font size

5. There are also icons to reopen the Help Navigator, to go back or forward through Help topics viewed in this session and to Print this topic.

INDEX TAB

1. This works like a pick list, opening topics or words in the whole Help system as you type letters into the index bar. As a word is exposed you can double-click on it to view the word in a Help topic.

SEARCH TAB

1. Working in a more defined way than the Index operation, the Search option allows for case sensitivity and Boolean searches, and you can click on the results to open the relevant Topic window.


Click on the X screen icon, or FILE >> Close (CTRL-W) or FILE >> Exit (ALT-F4) to leave the Help Navigator window.
