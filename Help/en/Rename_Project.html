RENAME A PROJECT

This PROJECT Menu option allows you to rename an existing HRE Project, retaining the database in its existing location.

1. Highlight the Project that you wish to rename. In this example, the Project is named "Immigrants". Only Projects previously accessed can be renamed

2. Note that you cannot rename a currently Open project (a popup advice message will appear) - it must be closed first

3. For checking purposes, a SUMMARY of the chosen project may be viewed before proceeding

4. In the space provided, enter the new name for this Project. Spaces, Numerals, Upper/Lower case letters and the usual naming characters are permitted. The new name must be a unique project name, regardless of its location

5. Click to RENAME the Project or CANCEL to not proceed

6. The Project has now been renamed, to "New Immigrants" in this example, and the Projects list will then be updated

7. The Project renaming process is only for user access and identification purposes, as the Filename ("Immigrants_to_AUS" in this case) remains unchanged on the computer.
