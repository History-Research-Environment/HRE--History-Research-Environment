HRE Project Summary

This screen allows the user to examine a summary of a current HRE Project.

1. The Project Summary is a brief overview of a selected HRE Project

2. It can be viewed as a direct option from the PROJECT Menu, or may be viewed for a chosen open Project from several of the action menus: Open Project, Backup Project, Close Project, Copy Project (brief Summary), Rename Project, Delete Project and Exit HRE (Close procedure)

3. The summary shows:

  • the current time
  • the Project Name as selected by the creator
  • the Filename used for storing the Project data
  • the name of the Folder where it is stored
  • the server/computer name
  • when the Project was last Closed
  • when the Project was last Backed up
  • the HRE Database type and Version.
