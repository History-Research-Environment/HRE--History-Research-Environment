Close an Open HRE Project

This option allows you to Close a project that is currently open in HRE.

1. To close any presently open Project, select PROJECT from the menu, and then select the CLOSE PROJECT option

2. The list of Open Projects appears, showing Project Name, Filename and whether it is a local project or one on a server

3. Highlight the name of the Project you wish to close. In the illustration the 'My Workshop' Project has been selected from the currently Open projects. If there is only one open project it is selected by default

4. Once you highlight the required Project, the name appears below, to enable checking

5. Note that at this point, the selected Project can also be subjected to other processes, including SUMMARY, VALIDATE or BACKUP, and there is an option to CANCEL the closure

6. If you then select CLOSE PROJECT and confirm the decision, the Project will be closed (there will be no confirmation notice if that option was selected in Settings). HRE remains open so another Project can be opened if desired

7. If the Closed Project was currently the Active Project then you can opt to make another Project Active.
