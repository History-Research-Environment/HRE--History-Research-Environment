Manage Source Elements

The Manage Source Elements screen allows you to manipulate Source Elements in all of the supported languages.
Initially it shows elements from the currently set HRE Data Language.

Double-clicking on a selected element allows you to edit the element name.

The 'Add' button allows creation of new Source Elements in the chosen language.

Note that if you add an element in your chosen language, there will be no corresponding element in any other language.
This may be corrected by use of the 'Translate' button, which allows you to add an equivalent element in another language.

Elements may be found in the Element list by entering text in the 'Find' text box.
