Project Administration

This screen allows a project administrator to define the users who may have access to an HRE project.
By default, the original creator of an HRE project has full administration rights to the project and can then use this function to add other users.

The screen displays a table of projects currently known to this HRE implementation, followed by a table of the known users, which is populated when a project is selected in the project table.
A project may also be searched for by use of the 'Browse...' button.

Controls are provided for Adding and Deleting users and changing the password of a user.
Passwords must be at least 8 characters, including a capital letter, lower case letter, number and a special character.
