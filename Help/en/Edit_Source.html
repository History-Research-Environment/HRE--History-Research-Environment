Add and Edit Sources

The screen for Adding/Edit Sources allows you to create or edit all Source data

The screen consists of 3 tabs, for General Source data, Additional information and this Source's Connections to sources and Repositories.

General tab:
This defines the source Title and abbreviated Title (both must be present) and the Source Type they are derived from.
Based on the selected Source type it also shows the Footnote and Bibliography templates that apply, plus all the relevant Source Elements belonging to those templates, with their assigned values (if any).
If a template has been edited to be applicable to only this Source (a 'local template') then it is identified as having overridden the global (Source Type) template.
The 'Preview' buttons allow you to see the result of substituting the current Source Element values into each of the templates.

Additional tab:
Under this tab the Source Fideility and its Active/Inactive state may be defined.
Other data that may be recorded includes the Source Reference and Reminder data.
If any person defined in the project is also an Author, Editor or Compiler of this Source data, they may also be recorded here.

Connections tab:
Allows for the addition (or deletion) of any Source of this Source and the linkage to a Repository for this Source.
