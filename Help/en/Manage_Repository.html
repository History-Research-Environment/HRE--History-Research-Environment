Manage Repository

The Manage Repository screen allows you to view, Edit, Copy Add and Delete Repository data.

Use of the 'Add' and 'Edit' functions displays the Repository detail screen, which allows for setting the Repository name, abbreviated name, location (according to the selected Loction Style) and a Memo.

A Repository that is in use may not be deleted.

Repositories may be found in the list by entering text in the 'Find' text box.
