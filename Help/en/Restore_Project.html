Restore a backed up HRE Project

This screen allows you to restore previous backups and then work with that file as the current Open Project.
The original Project, if still present, will only be overwritten if you specifically allow this to happen.

1. Click on FIND A BACKUP FILE and from the File Chooser select the file you wish to restore; alternatively, browse to any other location and SELECT the file

2. Your selection, including its descriptive name, will be transferred back to the Project Restore window. Check the NOTES as to the options from here

3. Set the folder location where the Backup is to be restored to, again from the File Chooser window

4. The file name defaults to that used by the original Project, but MUST be changed if the same destination is to be used so as not to overwrite the original file

5. It is essential that the restored project be given a new descriptive name to differentiate it from the original project. Once this has been entered, click RESTORE

6. When completed, the restored project will be the Active Project

7. On checking, if the restored project is what was required, you can later opt to delete the old version of the project if it is still on this computer.

8. The second part of the screen allows the restore of External Files (images, documents etc) from their backup zip files created in Project Backup. Note that the two parts of the screen work independently - in other words, you may restore an External File without restoring a Project file first.
