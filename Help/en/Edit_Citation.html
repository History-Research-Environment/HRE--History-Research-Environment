Edit Citations

The Edit Citation screen allows you to define and edit all Citation data.

This allows for defining the Citation's Detail, Memo and Reference data as well as defining Surety values for the linked person(s), date, location and memo data.

The Assessor of this Surety rating is displayed for reference and by default is the project's current 'Owner', as set under Tools->Edit Users and Rights->Defined Users.

Based on the Source for this citation (which may be selected), the screen displays the Source's Fidelity setting.

The 'Preview' button displays how the Footnote and Bibliographies look, using the Source and Citation's currently defined information.
A second click of the 'Preview' button hides this preview display.
