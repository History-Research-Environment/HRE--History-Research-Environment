EXIT from HRE

This option allows you to close Open Project(s) and exit from the HRE program, with an option to carry out other processes before the close.

There are two ways to Exit HRE, and there are slight variations, but the end result is the same.

A] FROM THE PROJECT MENU

1. Select Close Project from the main Project menu

2. Click on the Project you want to close, so it is highlighted

3. You now have five options
- To view a Summary of the selected Project
- To Validate the Project, ensuring indexes and other parts are correct and complete, and auto-correcting any errors if possible
- To Backup the Project before you close it, giving you an option to record your backup in a chosen location
- To Cancel the closure
- To actually Close the Project.

4. As each project is closed, a message will advise this has happened, unless Cancel prompts have been turned off in the HRE Settings

5. HRE closes when all open Projects have been closed

6. Note that the last project closed will be the project that is automatically opened at the next use of HRE if the HRE Setting to 'Open last closed project' has been turned on.


B] FROM THE X AT TOP RIGHT OF THE HRE MAIN SCREEN

1. You will be asked to confirm that you want to exit HRE, unless Cancel prompts have been turned off in the HRE Settings

2. If there is only one project open, it will be closed

3. If more than one project is open, the Close Project screen described above will open

4. As each project is closed, a message will advise this has happened, unless Cancel prompts have been turned off in the HRE Settings

5. HRE closes when all open Projects have been closed

6. Note that the last project closed will be the project that is automatically opened at the next use of HRE if the HRE Setting to 'Open last closed project' has been turned on.
