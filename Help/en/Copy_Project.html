COPY AN HRE PROJECT

This screen allows you to copy an HRE Project from any source to any chosen destination. It may then be opened as required. The default Copy location may be set in Tools > Settings > User > File Locations.

1. You may select a project from the Known Projects list or use BROWSE to open the File Chooser window to find an HRE file on this computer, on a remote source or a removable drive. Select the required Project

2. In COPY TO give the new project a name

3. Again use File Chooser to BROWSE for the Project destination folder and allocate the new file name

4. Press COPY for the process to take place

5. This Project may now be Opened on this computer or on another computer.
