OPEN AN HRE PROJECT

This screen allows you to open an existing HRE Project from any chosen location. It will then become the ACTIVE PROJECT.

HRE allows up to THREE Projects to be open at any time but only one may be nominated to be ACTIVE. This will be either the last Project opened OR the project selected by use of the "P" Project advice symbol at the end of the menu bar.

QUICK OPEN: recently used HRE Projects are shown as a list at the bottom of the Project menu, below the menu options. Recent Projects, identified by the HRE logo, may be opened by clicking on the Project name.


1. To Open any existing HRE Project, BROWSE to the database location if it is not shown in the default list. Projects may be local, on a network or a removable drive

2. The Selected Project's name and current file name will be displayed, and a SUMMARY may be viewed

3. Click on OPEN (or double-click the Project name) to open the Project, or CANCEL to exit this screen

4. Once a Project has been opened, it will automatically be made the Active Project

5. At any time, you may opt to change the Active Project.
