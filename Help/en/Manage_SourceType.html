Manage Source Types

The Manage Source Types screen allows you to view, Edit, Copy Add and Delete Source Type data.

When a Source Type is selected, the templates for its Footnote and Bibliography structures are also displayed.
To change these templates, make use of the 'Edit' button (or double-click the selected entry).

Use of the 'Add' and 'Edit' functions displays the editable Source Type screen, which allows editing of the templates and Reminder text.

A Source Type that is in use may not be deleted.

Source Types may be found in the list by entering text in the 'Find' text box.
