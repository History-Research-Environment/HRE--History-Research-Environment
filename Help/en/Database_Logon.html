Database Logon

This dialog appears when you attempt to open an HRE project defined on a remote server.

The screen displays 3 data entry areas, as follows:
1. Remote connection data, consisting of the Database type; server IP address; server port number. This information is mainly for confirmation purposes and you would normally not be expected to change it

2. Your logon ID for this project, as provided by the System Administrator of the remote server and saved by HRE

3. Your password for this project, as initially provided by the System Administrator of the remote server

NOTE:

* HRE will only allow 3 logon attempts before failing the logon process.

* Use of a remote server requires HRE to be in 'server mode' - this is set in Tools->Settings->Server Mode.
