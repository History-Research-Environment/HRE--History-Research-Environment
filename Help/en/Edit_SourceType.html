Edit Source Types

The Edit Source Types screen allows you to view and edit data for the selected Source Type.

You may change the Source Type Name and edit any of the three templates for footnotes and bibliography entries, plus the Reminder text.

You will receive error messages if you attempt to save unknown Source Element names or use invalid formatting codes such as [ITAL].
