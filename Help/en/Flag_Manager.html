Flag Manager

The Flag Manager displays the current status of all defined Flags and enables the user to Add, Edit and Delete Flags.

The screen displays a table of all flags, consisting of the Flag Name, whether it is a System-defined flag. the Flag's language, whether it is an Active Flag and all the possible settings defined for the Flag.

Controls for filtering the table contents sit above the Flag table. To filter the Flag list, enter the required text in the "Text to filter for:" field, select the "Filter" checkbox and Select the table column to use to apply the filter (or choose All Columns). Removing the "Filter" check will return the display to the full list.

On the left of the screen are controls for Adding or Deleting Flags and for moving Flags up or down in the display order.

Double-clicking on any Flag (or clicking the "Add a new Flag" button) displays a Flag edit screen.

The Flag Edit screen allows changing (or setting) the Flag's name, its Active status, the available Flag values and optionally a description of the use of the Flag and its values.

When adding or editing Flag details, you should be aware of the following rules:

1. System-defined flags cannot be added or deleted (but can be made inactive)

2. Inactive flags remain defined but are not displayed in a Person's flag list

3. The first flag value is always the default value of the flag

4. The order of a flag's possible settings cannot be changed. That is, settings such as ‘?, N, Y’ cannot be re-ordered to ‘?, Y, N’

5. Abbreviated flag values, like 'N' or 'Y' can be expanded to meaningful words (like 'No', 'Yes') up to TEN characters long but the first letter of the word cannot be changed. So you could set up a flag for Occupation with a value of Accountant and later change it to Actuary but you could not change Doctor to Surgeon.

6. After selecting a flag in the flag table, its position in the table may be moved up or down by use of the ‘MOVE UP/DOWN’ buttons. This applies to all flags, including System flags and Inactive flags.
