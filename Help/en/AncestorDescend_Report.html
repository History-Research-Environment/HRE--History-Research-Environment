Produce an Ancestor or Descendant Report

This screen allows the user to view and print a restricted report of one or more generations of ancestor or descendants from a chosen focus person

1. Using the Person Selector (from PERSON >> Select Person by), locate the target person for the report and note the Person Number

2. Select the Ancestor/Descendant Report from the REPORTS screen

3. Enter the Person Number into that box, and press [ENTER]. The person will appear in the display space with his/her Ancestors (default) displayed.

4. You can now display ancestor details manually (click on the plus '+' sign) or use the EXPAND ALL or EXPAND 1 GENERATION buttons
Note that by double-clicking any generation, only that generation is expanded, so you can be selective as to which generations are fully displayed. Similarly, by use of the minus '-' sign or COLLAPSE ALL or COLLAPSE 1 GENERATION the display can be further varied

5. If you select to show Descendants, the same expand and collapse options apply

6. If a printed copy is required, a Header and/or Footer can be added using their selection boxes. Selecting PRINT will open to default settings for the page set up, and then progress to printing with your default or other chosen printer

IMPORTANT NOTE: If you use either a Header or Footer, then the printed report will only show that part of the tree that is visible on screen
The tree area can be widened or made longer by expanding the screen to allow more names and details to be included in view

7. Printing without a Header or Footer will reproduce all text, whether in view on screen or not

8. Entering the number of another person and pressing [ENTER] will change the display to focus on that person

9. Select CLOSE to exit back to the HRE main display.
