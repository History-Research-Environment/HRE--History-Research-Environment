Manage Sources

The Manage Sources screen displays a list of the defined Sources and how many times they have been Cited.
The screen allows you to view, Edit, Copy Add and Delete Source data, but a Source that is in use may not be deleted.

When a Source is selected, the templates for its Footnote and Bibliography structures are also displayed.
To change these templates, make use of the 'Edit' button (or double-click the selected entry).

Use of 'Add' button allows creation of a new Source and 'Edit' allows manipulation of a currect Source's definition.

The 'Show Inactive' button displays only Sources that have been marked as inactive, but may still be present in the database.

Sources may be found in the list by entering text in the 'Find' text box.
