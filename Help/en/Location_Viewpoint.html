LOCATION VIEWPOINT

The Location Viewpoint allows the user to see all the information recorded in HRE for the selected location. Up to 5 Location Viewpoints may be opened for each open HRE project
The screen initially opens in a collapsed format showing only the location's name and 6 sub-screen title bars.

The 6 sub-screens defined by the title bars lie to the left and right of a vertical divider bar. The divider may be moved left or right to change the amount of screen space allocated to each half of the screen. At the top of the divider are two small arrow buttons which may be used to make either side take over the whole screen.

Each of the sub-screens can be displayed by clicking on the red triangle at the left of each title bar, which changes the triangle to a green downward-pointing format and opens the sub-screen. Clicking on a green triangle collapses the sub-screen. Each title bar also indicates the number of entries contained in the sub-screen, such as "Events (6)".

When a sub-screen is displayed it may also show "+" and "-" buttons at the top right. These enable expansion or contraction of the number of rows of data displayed in the sub-screen.

Should the height of the Viewpoint screen become larger than the enclosing HRE screen, the Viewpoint screen will change to a scrollable format.

Events sub-screen:
Each entry shows information about an event occurring at this location. A checkbox enables restricting the list to show only events for which there was a Key Associate.

Superior/Inferior sub-screen:
This is not yet implemented in HRE.

Associates sub-screen:
This displays a list of all people associated with the selected location through their involvement in events at this location.

Images sub-screen:
Displays all images (if any) of the location.

Flags sub-screen:
Displays all Active flags for the location and their current settings.

Notepads sub-screen:
Notepads are not yet implemented in HRE.
