Manage Event

The Manage Event screen may be invoked in two ways, for different purposes.

1) When invoked from the main menu it allows for the creation of new Events, Editing, Deleting or Copying an Event, or marking it as Disabled.
An Event that is in use in your project may not be deleted.

2) When invoked from a Person's screen by use of 'Add Event' it allows you to select events for use.
This requires selection of the required Event and Role. The other functions described above may also be performed at this time.

By default the screen shows a list of All Events. Selecting an Event Group (like Birth, Marriage, etc) restricts the displayed events to just those events belonging to the selected group.

Selecting 'Your Event Groups' changes the display to a different list which starts with the groups Census, Travel, Military and includes 4 groups whose names are undefined but which may be used as you wish.

Selecting 'Standard Groups' returns you to the first group list.

The "Other Events' selection shows all events not assigned to a particular group.

Events can be quickly found by typing into the 'Find' text-box when 'All Events' are displayed.
