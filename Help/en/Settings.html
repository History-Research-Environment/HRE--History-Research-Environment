HRE Settings

This screen allows you to set a number of preferences for how HRE operates. The settings can be changed at any time and are not affected by closing and reopening HRE or your Projects.

1. From the main menu TOOLS, select SETTINGS, then USER to select and set a range of User options.

2. From the Settings set of Tabs, the user can choose to change:

HRE Look and Feel: to set the 'look and feel' - the theme/font size and type (the HRE default is Graphite, Blue-medium font, Arial)

Language Settings: to set the default languages for the HRE screens, Data and Reports

Format Settings: to set the Date and Time formats

User Options: to select showing of the Welcome Screen, Cancel messages, Opening of the last project saved at HRE re-start, whether to Backup a Project as it closes and whether HRE Plugins should be enabled

Accents: to set the colors of font and background for any HRE Themes you wish to define

File Locations: to set the default locations for HRE Projects, Reports, Backups and External File Backups

Debug Options: to set the writing of a Log file (default), the writing of DEBUG messages and timing messages. These options are primarily for use by HRE Technical Support

Server Mode: to enable execution of HRE in Server Mode (default is OFF)
