CREATE A NEW PROJECT

This screen allows you to create a new project, set its name and filename and nominate where the database files are to be stored.

Note that this is NOT to be used to Import a TMG data set. The process for Importing data from TMG is separately selected from the TOOLS Menu.

Selecting Create Project from the Project Menu provides a three step process to follow:

1. Give your Project a name to identify it in future. This should be descriptive and may consist of several words, but must be unique

2. Click on BROWSE to choose where the database is to be stored. A default location will be presented but you may choose another if required

3. Once the file location has been selected, give your Project a concise file name, which must be different to any other HRE Project file stored in that location. Windows/MAC conventions as to characters you can use in filenames must be followed and HRE will automatically add the .mv.db file suffix. Press SELECT.

Now the Window will be complete showing your new Project Name, Filename and Location

At this point you can opt to CANCEL or CREATE the new Project.

If you have now created the project it will then be initialised, opened and made ACTIVE. A window will appear to signify that process has been completed.

You may now begin entering data into the new Project, or opt to change to another open project to be Active, or Open another project.
