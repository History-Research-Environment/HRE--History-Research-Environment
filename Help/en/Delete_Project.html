DELETE A PROJECT

This screen allows the user to delete a surplus Project.

Note that the project cannot be currently open in HRE - it must be closed first.

1. Select the project you wish to Delete. The Project Name, Project Filename and location will then be filled in below the selection chart

2. To be sure, checking the Project SUMMARY may be wise at this time

3. If proceeding, press DELETE

4. Keep Database: You will be prompted as to whether you wish to keep the database file (while deleting the HRE Project reference), or to delete both. Answer NO to delete all, or YES to keep the database file.
