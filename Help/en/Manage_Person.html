Manage (Edit) a Person

The Manage Person screen allows you to view and modify all existing data related to a person.
The top section of the screen shows the identity of the person, Reference data and Birth-Sex, Partner and Children counts.

Below that is the Parents table - this shows data for ALL related parents of any type (biological, adopted, foster, etc).
Right-clicking within the Parents table allows you to choose to delete, add or edit these entries.

Edit Categories

The left panel of the screen allows you to choose to display data from 8 different categories, starting with Events, which are displayed by default when this screen opens.
Each of these categories is explained in the following:

Events
* This table lists all recordeed events in order of their Sort Date
* Witnessed events and Child events are shaded and can be removed or included in the event list by use of the check-boxes at the top of the list
* Right-clicking in the table of events shows a list of possible actions, namely editing or deleting the selected event or adding a new event
* A right-click in a Name-type event automatically opens an edit screen for that Name event.

Partners
* This table displays a list of all partners of this person
* Care must be taken in the definition and creation of a person's partners and their related events, as HRE allows you to add a Partner without creation of an event linking the two people
* You may add a partner from a right-click in the Partner table, then selecting 'Add new Partner'
* After selecting someone from the offered list you can complete the process by defining the Event-type, Roles, optionally a Memo and then selecting whether you wish to add this Partner with, or without, also defining a Partner Event
* Note: If a partner is created without a connecting event, they do not apear in either person's Associate table, as there is no 'association' between them.

Associates
* This table displays a list of all Associates of this person
* Each entry shows the Associate, the event and date, and that person's Role in that event
* A filter may be applied to filter for any text in any or all columns of the table.

All Names
* This table displays a list of all the person's names, with the Primary name followed by the word 'Primary' in brackets
* Right-clicking in the table sdisplays an Add/Delete/Edit menu; double-click of an entry opens the edit screen for that name.

Flags
* This table displays a list of all defined Flags and their settings for this person
* Any of the current settings may be changed by clicking on the 'Setting' of a Flag.

Media
* This displays all media (images, text, audio or video) attached to this person, with captions.

Notepads and DNA
* These categories have not yet been implemented.
