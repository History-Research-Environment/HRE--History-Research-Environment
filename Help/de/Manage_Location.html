Standort verwalten

Im Bildschirm „Standort verwalten“ können Sie die verschiedenen Elemente bearbeiten, die die Daten eines Standorts definieren, entsprechend Ihrer Auswahl des Standortstils.

Sie können auch einen Datumsbereich festlegen, um den Zeitraum zu definieren, in dem dieser Standort unter diesem Namen bekannt war.

Im Abschnitt „Medien“ können Sie relevante Medien hinzufügen, um diesen Standort zu beschreiben.
