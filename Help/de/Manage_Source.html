Quellen verwalten

Der Bildschirm „Quellen verwalten“ zeigt eine Liste der definierten Quellen und deren Zitationshäufigkeit an.
Sie können Quellendaten anzeigen, bearbeiten, kopieren, hinzufügen und löschen. Eine Quelle, die bereits verwendet wird, kann jedoch nicht gelöscht werden.

Wenn Sie eine Quelle auswählen, werden auch die Vorlagen für ihre Fußnoten- und Bibliografiestruktur angezeigt.
Um diese Vorlagen zu ändern, verwenden Sie die Schaltfläche „Bearbeiten“ (oder doppelklicken Sie auf den ausgewählten Eintrag).

Mit der Schaltfläche „Hinzufügen“ können Sie eine neue Quelle erstellen, und mit „Bearbeiten“ können Sie die Definition einer bestehenden Quelle bearbeiten.

Die Schaltfläche „Inaktive anzeigen“ zeigt nur Quellen an, die als inaktiv markiert wurden, aber möglicherweise noch in der Datenbank vorhanden sind.

Sie können Quellen in der Liste finden, indem Sie Text in das Suchfeld eingeben.
