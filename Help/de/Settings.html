HRE Einstellungen

In diesem Bildschirm können Sie eine Reihe von Einstellungen für die Arbeitsweise von HRE vornehmen. Die Einstellungen können jederzeit geändert werden und werden durch das Schließen und erneute Öffnen von HRE oder Ihrer Projekte nicht beeinflusst.

1. Wählen Sie im Hauptmenü WERKZEUGE die Option EINSTELLUNGEN und dann BENUTZER, um eine Reihe von Benutzeroptionen auszuwählen und einzustellen.

2. Auf den Registerkarten können Sie Folgendes ändern:

    HRE Look and Feel: Um das "Look and Feel" einstellen - das Thema / Schriftgröße und -typ (die HRE-Standardeinstellung ist Graphite, Blue-medium font, Arial)

    Spracheinstellungen: So legen Sie die Standardsprachen für die HRE-Bildschirme, Daten und Berichte

    Formateinstellungen: So stellen Sie die Datums- und Uhrzeitformate

    Benutzeroptionen:

  • Begrüßungsbildschirm beim HRE-Start anzeigen (Standard: Nein)
  • Bestätigungsanfragen für Abbrechen und andere Aktionen anzeigen (Standard: Ja)
  • Das zuletzt geschlossene HRE-Projekt beim HRE-Start öffnen (Standard: Nein)
  • Projekt sichern, wenn es geschlossen wird (Standard: Nein)
  • Plugin-Code verarbeiten (Standard: Nein)
  • Die Personenauswahl wird immer neu geladen, wenn eine Person zur Datenbank hinzugefügt wird (Standard: Ja). Ist diese Option deaktiviert, wird die Personenauswahl nur neu geladen, wenn Sie sie über das HRE-Menü aufrufen.
  • Beim Speichern von Eheschließungen immer nach dem Ehenamen fragen (Standard: Nein)

    Akzente: Um die Farben von Schriftart und Hintergrund für alle HRE-Themen festzulegen, die Sie definieren möchten

    Dateiorte: So stellen Sie die Standardorte für HRE-Projekte, Berichte, Backups und externe Dateisicherungen

    Debug-Optionen: Um das Schreiben einer Protokolldatei (Standardeinstellung), das Schreiben von Debug-Nachrichten und Timing-Nachrichten einzustellen. Diese Optionen sind hauptsächlich für den Einsatz von HRE Technical Support

    Server-Modus: Um die Ausführung von HRE im Servermodus zu aktivieren (Standardeinstellung ist Nein)

Verwandte Themen

  • Projekt Öffnen
  • Importieren eines Projekts von TMG
  • Projekt-Zusammenfassung
  • Projekt Wiederherstellen
  • Projekt Schließen
  • Projekt Kopieren
  • Projekt Umbenennen
  • Projekt Löschen
