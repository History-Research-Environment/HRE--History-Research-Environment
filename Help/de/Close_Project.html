Schließen eines offenen HRE-Projekts

Mit dieser Option können Sie ein Projekt schließen, das derzeit in HRE geöffnet ist.

1. Um ein aktuell geöffnetes Projekt zu schließen, wählen Sie im Menü PROJEKT und dann die Option PROJEKT SCHLIESSEN

2. Die Liste der geöffneten Projekte wird angezeigt, mit Projektname, Dateiname und der Angabe, ob es sich um ein lokales Projekt oder ein Projekt auf einem Server handelt

3. Markieren Sie den Namen des Projekts, das Sie schließen möchten. In der Abbildung wurde das Projekt "My Workshop" unter den derzeit geöffneten Projekten ausgewählt. Wenn nur ein Projekt geöffnet ist, wird dieses standardmäßig ausgewählt.

4. Sobald Sie das gewünschte Projekt markiert haben, erscheint der Name darunter, um eine Überprüfung zu ermöglichen

5. Beachten Sie, dass das ausgewählte Projekt an dieser Stelle auch anderen Prozessen unterworfen werden kann, einschließlich ZUSAMMENFASSUNG, ÜBERPRÜFEN oder SICHERN, und dass es eine Option für das ABBRECHEN des Schließvorgangs gibt

6. Wenn Sie dann PROJEKT SCHLIESSEN wählen und die Entscheidung bestätigen, wird das Projekt geschlossen (es erfolgt kein Bestätigungshinweis, wenn diese Option in den EINSTELLUNGEN gewählt wurde). HRE bleibt geöffnet, so dass ein anderes Projekt geöffnet werden kann, falls gewünscht

7. Wenn das geschlossene Projekt derzeit das aktive Projekt war, können Sie ein anderes Projekt aktiv machen.

Verwandte Themen

  • Projekt Öffnen
  • Projekt-Zusammenfassung
  • Projekt Sichern
  • Projekt Kopieren
  • Projekt Umbenennen
  • Projekt Löschen
