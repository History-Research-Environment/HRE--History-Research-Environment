Was wollen Sie als nächstes tun?

An diesem Punkt gibt es eine Reihe von Aufgaben, die Sie möglicherweise ausführen möchten. Eine Liste dieser Aufgaben und wie Sie sie ausführen, wird in den folgenden Abschnitten behandelt.

HRE für Ihre Verwendung einrichten: Wir empfehlen Ihnen dringend, zunächst unter "Werkzeuge -> Einstellungen -> Benutzer" die HRE-Einstellungen anzupassen, insbesondere Ihre bevorzugten Einstellungen für den Dateispeicherort.

So öffnen Sie das Beispielprojekt und erkunden es: Wählen Sie im Menü "Projekt" die Option "Projekt öffnen" und wählen Sie dann das Beispiel aus. Klicken Sie auf "Öffnen", oder doppelklicken Sie einfach auf den Beispiel-Eintrag.

So öffnen Sie eines Ihrer Projekte: Wenn es sich in der Liste "Letzte Projekte" im Menü "Projekt" befindet, klicken Sie es einfach an. Andernfalls wählen Sie "Projekt öffnen" und wählen dann Ihr Projekt aus. Klicken Sie auf "Öffnen", oder doppelklicken Sie einfach auf den gewünschten Projekteintrag.

So erstellen Sie ein neues leeres Projekt für Ihre Daten: Wählen Sie im Menü "Projekt" die Option "Neues Projekt erstellen" und folgen Sie den Anweisungen, um den Namen und den Speicherort festzulegen.

So finden und öffnen Sie ein Projekt, das Sie von jemand anderem erhalten haben: Wählen Sie im Menü "Projekt" die Option "Projekt öffnen" und klicken Sie dann auf "Durchsuchen", um die angegebene Datei zu finden. Klicken Sie dann auf 'Öffnen'.

So stellen Sie ein zuvor gesichertes Projekt wieder her: Wählen Sie im Menü "Projekt" die Option "Projekt wiederherstellen" und folgen Sie den Aufforderungen.
