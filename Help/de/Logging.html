HRE Protokollierungs-Bericht

Dieser Bildschirm ermöglicht es dem Benutzer, die HRE-Aktivitätsprotokolldateien zu untersuchen, die bei jedem Öffnen von HRE erstellt werden, und sich dafür zu entscheiden, verschiedene Aufgaben für diese Dateien auszuführen.

1. Wählen Sie BERICHTE >> PROTOKOLLIERUNG

2. Klicken Sie auf DURCHSUCHEN, um den Standardordner für HRE-Protokolldateien zu öffnen, wie gezeigt

3. Eine Liste der täglichen Protokolldateien ist nun zu sehen, und ein Doppelklick auf eine dieser Dateien öffnet die Datei, die alle HRE-Aktivitäten während des jeweiligen Tages zeigt

4. Die Protokolldatei kann eine große Menge an Daten unter den Überschriften enthalten: Datum, Uhrzeit, Server (Computer), Projektname und Protokolleintrag. Beachten Sie, dass die letzte Spalte sowohl Programmaktionen als auch Fehler enthalten kann

5. Sie können nun den Inhalt des Protokolls oder einen ausgewählten Teil davon in eine Datei oder auf einen Drucker ausgeben. Dies kann für den technischen Support oder zur Rückverfolgung von Schritten, die in einer HRE-Sitzung gemacht wurden, nützlich sein. Um Alles oder eine Auswahl auszugeben, markieren Sie mit der Maus die gewünschten Zeilen des Protokolls. Das Ziel kann eine Datei oder ein Drucker sein, auch eine PDF-Datei. Wenn Sie das getan haben, können Sie durch Klicken auf Auswahl zurücksetzen die gesamte Protokolldatei wieder anzeigen lassen oder eine andere Protokolldatei auswählen.

6. Schließen Sie das Fenster Protokollierung, wenn Sie fertig sind.

Verwandte Themen

  • Projekt Öffnen
  • Projekt-Zusammenfassung
  • Projekt Schließen
  • Projekt Kopieren
  • Projekt Umbenennen
  • Projekt Löschen
