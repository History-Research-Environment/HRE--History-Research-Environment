Repository verwalten

Im Bildschirm „Repository verwalten“ können Sie Repository-Daten anzeigen, bearbeiten, kopieren, hinzufügen und löschen.

Durch die Verwendung der Funktionen „Hinzufügen“ und „Bearbeiten“ wird der Detailbildschirm des Repositorys angezeigt. Hier können Sie den Repository-Namen, die Kurzbezeichnung, den Standort (entsprechend dem ausgewählten Standortstil) und eine Notiz festlegen.

Ein Repository, das bereits verwendet wird, kann nicht gelöscht werden.

Sie können Repositories in der Liste finden, indem Sie im Textfeld „Suchen“ einen Text eingeben.

Verwandte Themen

  • Zitate bearbeiten
  • Quellen verwalten
  • Quellenelemente verwalten
  • Quellentypen verwalten
  • Quellen bearbeiten
  • Quellentypen bearbeiten
  • Repositories bearbeiten
