Sichern eines HRE Projekts

In diesem Bildschirm können Sie ein vorhandenes HRE-Projekt sichern, unabhängig davon, ob das Projekt gerade in HRE geöffnet ist oder nicht. Sie können das Ziel für die Sicherungsdatei auswählen und festlegen, ob auch externe Ordner, die Teil des Projekts sind, gesichert werden sollen oder nicht.

1. Wählen Sie PROJEKT >> PROJEKT SICHERN und markieren Sie das gewünschte Projekt in der angezeigten Liste

2. HRE zeigt nun den Dateinamen an, der für die Sicherung verwendet wird, d. h. den Projektnamen mit der aktuellen Uhrzeit und dem Datum. An dieser Stelle gibt es eine Option, um eine ZUSAMMENFASSUNG des ausgewählten Projekts anzuzeigen

3. Klicken Sie auf DURCHSUCHEN, um den Ordner zu finden, in dem die Sicherung gespeichert werden soll. Obwohl der Standardordner angezeigt wird (wie in den HRE-Einstellungen für Dateispeicherorte definiert), können Sie durchblättern, um einen anderen Speicherort auszuwählen. Die Datei erhält automatisch das Suffix .hrez für kompakte Sicherungsdateien. Sobald der gewünschte Ordner im Fenster angezeigt wird, drücken Sie die Eingabetaste, um ihn auszuwählen.
Der Standard-Sicherungsort kann unter Werkzeuge > Einstellungen > Benutzer > Datei-Speicherorte festgelegt werden.

4. Sie können nun festlegen, dass auch externe Dateiordner (Bilder, Dokumente, Tondateien, Videos usw.), die mit diesem Projekt verbunden sind, gesichert werden sollen, indem Sie das Auswahlfeld markieren. Dadurch wird ein neues Fenster geöffnet, in dem Sie einen oder mehrere Ordner für die Sicherung auswählen und angeben können, wo sie gespeichert werden sollen, was sich vom Speicherort des Projekts unterscheiden kann. Der Dateiname der Sicherung enthält den Namen des Originalordners und ist ein Standard-Zip-Dateityp.

5. Klicken Sie nun auf SICHERN, um den Vorgang zu starten. Wenn das angegebene Projekt gerade in HRE geöffnet ist, wird es zuerst geschlossen und dann wird die Sicherung gestartet. Die Projektdatenbank wird zuerst gesichert, gefolgt von den externen Dateiordnern (falls Sie welche angegeben haben).

6. Wählen Sie alternativ ABBRECHEN, wenn die Sicherung nicht mehr erforderlich ist.

Verwandte Themen

  • Projekt Öffnen
  • Projekt-Zusammenfassung
  • Projekt Schließen
  • Projekt Kopieren
  • Projekt Umbenennen
  • Projekt Löschen
