UMBENENNEN EINES PROJEKTS

Mit dieser PROJEKT-Menüoption können Sie ein bestehendes HRE-Projekt umbenennen, wobei die Datenbank an ihrem bisherigen Speicherort verbleibt.

1. Markieren Sie das Projekt, das Sie umbenennen möchten. In diesem Beispiel hat das Projekt den Namen "Immigrants". Nur zuvor aufgerufene Projekte können umbenannt werden

2. Beachten Sie, dass Sie ein aktuell geöffnetes Projekt nicht umbenennen können (es erscheint eine Popup-Hinweis-Meldung) - es muss zuerst geschlossen werden

3. Zur Kontrolle kann vor dem Fortfahren eine ZUSAMMENFASSUNG des gewählten Projekts angesehen werden

4. Geben Sie in dem dafür vorgesehenen Feld den neuen Namen für dieses Projekt ein. Leerzeichen, Ziffern, Groß-/Kleinbuchstaben und übliche Dateinamenszeichen sind erlaubt. Der neue Name muss ein eindeutiger Projektname sein, unabhängig von seinem Standort

5. Klicken Sie auf UMBENENNEN, um das Projekt umzubenennen, oder auf ABBRECHEN, um nicht fortzufahren

6. Das Projekt wurde nun umbenannt, in diesem Beispiel in "New Immigrants", und die Projektliste wird anschließend aktualisiert

7. Der Prozess der Projektumbenennung dient nur dem Benutzerzugang und der Identifizierung, da der Dateiname (in diesem Fall "Immigrants_to_AUS") auf dem Computer unverändert bleibt.

Verwandte Themen

  • Importieren eines Projekts von TMG
  • Projekt-Zusammenfassung
  • Projekt Sichern
  • Projekt Wiederherstellen
  • Projekt Schließen
  • Projekt Kopieren
  • Projekt Löschen
