EIN NEUES PROJEKT ERSTELLEN

In diesem Bildschirm können Sie ein neues Projekt erstellen, seinen Namen und Dateinamen festlegen und angeben, wo die Datenbankdateien gespeichert werden sollen.

Beachten Sie, dass dies NICHT für den Import eines TMG-Datensatzes verwendet werden soll. Das Verfahren zum Importieren von Daten aus TMG wird separat aus dem WERKZEUGE-Menü ausgewählt.

Auswählen von "Projekt erstellen" aus dem Menü "Projekt" bietet einen dreistufigen Prozess, der zu befolgen ist:

1. Geben Sie Ihrem Projekt einen Namen, um es in Zukunft zu identifizieren. Dieser sollte beschreibend sein und kann aus mehreren Wörtern bestehen, muss aber eindeutig sein

2. Klicken Sie auf DURCHSUCHEN, um zu wählen, wo die Datenbank gespeichert werden soll. Es wird ein Standardspeicherort angezeigt, aber Sie können bei Bedarf einen anderen wählen

3. Nachdem Sie den Speicherort für die Datei ausgewählt haben, geben Sie Ihrem Projekt einen prägnanten Dateinamen, der sich von allen anderen HRE-Projektdateien unterscheiden muss, die an diesem Speicherort gespeichert sind. Die Windows/MAC-Konventionen bezüglich der Zeichen, die Sie in Dateinamen verwenden können, müssen eingehalten werden, und HRE fügt automatisch die Dateiendung .mv.db hinzu. Drücken Sie AUSWÄHLEN.

Jetzt ist das Fenster vollständig und zeigt Ihren neuen Projektnamen, Dateinamen und Speicherort an

An dieser Stelle können Sie wählen, ob Sie das neue Projekt ABBRECHEN oder ERSTELLEN wollen.

Wenn Sie nun das Projekt erstellt haben, wird es initialisiert, geöffnet und AKTIV gemacht. Es erscheint ein Fenster, das anzeigt, dass der Vorgang abgeschlossen ist.

Sie können nun mit der Eingabe von Daten in das neue Projekt beginnen oder zu einem anderen geöffneten Projekt wechseln, um aktiv zu werden, oder ein anderes Projekt öffnen.

Verwandte Themen

  • Projekt Öffnen
  • Projekt-Zusammenfassung
  • Projekt Sichern
  • Projekt Schließen
  • Projekt Kopieren
  • Projekt Umbenennen
  • Projekt Löschen
