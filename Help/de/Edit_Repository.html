Repository bearbeiten

Im Fenster „Repository bearbeiten“ können Sie die Daten des ausgewählten Repositorys anzeigen und bearbeiten.

Dies umfasst den Repository-Namen, die Kurzbezeichnung, den Standort (entsprechend dem ausgewählten Standortstil) und die Notiz.

Verwandte Themen

  • Zitate bearbeiten
  • Quellen verwalten
  • Quellenelemente verwalten
  • Quellentypen verwalten
  • Repositories verwalten
