Erzeugen eines Vorfahren- oder Nachkommenberichts

Dieser Bildschirm ermöglicht es dem Benutzer, einen eingeschränkten Bericht über eine oder mehrere Generationen von Vorfahren oder Nachkommen einer ausgewählten Fokusperson anzuzeigen und zu drucken

1. Suchen Sie mit dem Personen-Selektor (aus PERSON >> Person auswählen) die Zielperson für den Bericht und notieren Sie die Personennummer

2. Wählen Sie den Vorfahren-/Nachkommenbericht im Bildschirm BERICHTE

3. Geben Sie die Personennummer in dieses Feld ein, und drücken Sie die Eingabetaste. Die Person erscheint im Anzeigebereich, wobei ihre Vorfahren (Standard) angezeigt werden.

4. Sie können nun die Vorfahrendetails manuell anzeigen (klicken Sie auf das Pluszeichen '+') oder die Schaltflächen ALLE ERWEITERN oder 1 GENERATION ERWEITERN verwenden
Beachten Sie, dass durch einen Doppelklick auf eine beliebige Generation nur diese Generation erweitert wird, so dass Sie selektiv auswählen können, welche Generationen vollständig angezeigt werden. In ähnlicher Weise kann die Anzeige mit dem Minuszeichen '-' oder mit ALLE ZUKLAPPEN oder 1 GENERATION ZUKLAPPEN weiter variiert werden

5. Wenn Sie auswählen, dass Nachkommen angezeigt werden sollen, gelten die gleichen Optionen zum Erweitern und Einklappen

6. Wenn Sie die Nummer einer anderen Person eingeben und die Eingabetaste drücken, wird die Anzeige auf diese Person fokussiert

7. Wählen Sie SCHLIESSEN um zurück zur HRE-Hauptanzeige zu gelangen.

Verwandte Themen

  • Projekt Öffnen
  • Projekt-Zusammenfassung
  • Projekt Schließen
  • Projekt Kopieren
  • Projekt Umbenennen
  • Projekt Löschen
