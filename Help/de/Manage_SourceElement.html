Quellelemente verwalten

Im Bildschirm „Quelltextelemente verwalten“ können Sie Quelltextelemente in allen unterstützten Sprachen bearbeiten.
Anfangs werden Elemente der aktuell eingestellten HRE-Datensprache angezeigt.

Durch Doppelklicken auf ein ausgewähltes Element können Sie dessen Namen bearbeiten.

Mit der Schaltfläche „Hinzufügen“ können Sie neue Quelltextelemente in der gewählten Sprache erstellen.

Beachten Sie, dass beim Hinzufügen eines Elements in Ihrer gewählten Sprache kein entsprechendes Element in einer anderen Sprache vorhanden ist.
Dies lässt sich mit der Schaltfläche „Übersetzen“ beheben, mit der Sie ein entsprechendes Element in einer anderen Sprache hinzufügen können.

Elemente können in der Elementliste gefunden werden, indem Sie Text in das Textfeld „Suchen“ eingeben.
