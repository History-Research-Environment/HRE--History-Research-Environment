IMPORTIEREN EINES TMG-PROJEKTS IN HRE

HRE kann ein Projekt direkt aus der Datenbank von The Master Genealogist (TMG) Version 8.05 oder 9.x importieren.

VORBEREITUNG VON TMG:

In TMG sollte eine Reihe von Prozessen durchgeführt werden, um einen erfolgreichen Import zu ermöglichen. Diese sind:

1. Merkmale: Klicken Sie im TMG-Menü "Datei" auf den Eintrag "Merkmal-Verwaltung". Wenn er sich öffnet, schließen Sie einfach den Bildschirm. Diese einfache Aktion wird eine Reihe von Unstimmigkeiten beheben, die in den TMG-Merkmaleinträgen vorhanden sein können.

2. Miniaturansichten von Exponaten: Unter bestimmten Umständen können TMG-Miniaturansichten von Exponaten verloren gehen oder beschädigt werden und nicht in HRE importiert werden. Um dieses Problem zu beheben, gehen Sie wie folgt vor: Öffnen Sie ein beliebiges TMG-Exponate-Verzeichnis mit einem Eintrag darin und klicken Sie mit der rechten Maustaste auf das Exponat. Wählen Sie "Alle Miniaturen aktualisieren". Beachten Sie, dass dieser Vorgang bei einer großen Anzahl von Exponaten einige Zeit in Anspruch nehmen kann.


BEVOR SIE EINEN IMPORT STARTEN:

Überprüfen Sie, ob das TMG-Projekt nur einen Datensatz enthält - der Importvorgang wird nur den ersten Datensatz eines TMG-Projekts importieren.
Überprüfen Sie den Speicherort Ihrer TMG-Projektdatei (.pjc), wie er in TMG angegeben ist: Dateien >> Einstellungen >> Aktuelles Projekt >> Allgemein, damit Sie ihn schnell finden können, wenn Sie den Importvorgang starten.
Entscheiden Sie, wo Sie die HRE-Datenbank nach dem Import speichern wollen. Wenn Sie nicht unbedingt den Standard-Speicherort von HRE verwenden wollen, erstellen Sie einen Ordner (z.B. HRE-Projekte) an dem gewünschten Ort, und unter HRE >> Werkzeuge >> Einstellungen >> Benutzer >> Speicherorte können Sie diesen Ordner als Standard festlegen.


DER IMPORT VORGANG:

1. Wählen Sie "Werkzeuge", dann "Import" und die Option "Von TMG" erscheint. Klicken Sie auf diesen Menüpunkt

2. Geben Sie den Projektnamen ein, den Sie für dieses Projekt in HRE verwenden möchten.

3. Klicken Sie in das zweite Feld, um den SPEICHERORT auszuwählen, an dem Sie das neue Projekt speichern möchten. Standardmäßig ist dies die aktuelle Einstellung für HRE-Dateispeicherorte, aber Sie können diesen Speicherort bei Bedarf über die normale Datei- und Ordnerverwaltung ändern. Geben Sie den Dateinamen ein, den Sie für dieses Projekt verwenden möchten

4. Wählen Sie dann den Speicherort Ihrer TMG-Datenbank aus, indem Sie auf deren .pjc-Datei klicken

5. Sobald dies geschehen ist, beginnt der Importvorgang

6. Ein Fortschrittsfenster erscheint, während HRE die ausgewählte TMG-Datenbank durchläuft und sie als neues HRE-Projekt in HRE einliest.

7. Wenn Sie auf Schließen klicken, stehen die TMG-Daten für die Verwendung durch HRE zur Verfügung.

8. Verwenden Sie den Menüpunkt Projekt "Projekt Öffnen", um das Projekt zu öffnen

9. Dies könnte ein günstiger Zeitpunkt sein, um dieses neue HRE-Projekt zu sichern, bevor eine Bearbeitung vorgenommen wird.

HINWEIS: Die TMG-Datenbank wird dabei in keiner Weise verändert. Sie verbleibt in ihrem ursprünglichen Ordner und ist weiterhin in TMG verfügbar.


FEHLER IM IMPORTPROZESS:

Sollte der Importvorgang aufgrund von Fehlern in der TMG-Datenbank, die HRE nicht verarbeiten kann, fehlschlagen, kann der HRE-Support Sie bitten, weitere Informationen zur Fehlersuche mit Hilfe des folgenden Verfahrens bereitzustellen.
Bevor Sie fortfahren, löschen Sie die möglicherweise unvollständige HRE-Datenbank, die Sie soeben erstellt haben, indem Sie den Menüpunkt "Projekt löschen" unter dem Menüpunkt "Projekt" von HRE verwenden. Auf die Frage, ob Sie die Datenbank behalten wollen, antworten Sie mit 'Nein'.

Gehen Sie nun zu HRE >> Werkzeuge >> Einstellungen >> Benutzer >> Debug-Optionen und klicken Sie auf 'DEBUG-Meldungen schreiben', dann übernehmen Sie die Änderung.

Gehen Sie zu Schritt 1 des obigen TMG-IMPORTPROZESSES. Der Importbildschirm hat sich geändert und zeigt nun 6 Kontrollkästchen und den Ort an, an dem eine neue Protokolldatei erstellt werden soll.

Aktivieren Sie die Kontrollkästchen, die Sie vom HRE-Support erhalten haben, und wiederholen Sie die Schritte des TMG-Importprozesses. Wenn der Importvorgang erneut fehlschlägt, schließen Sie HRE und senden Sie dem Support eine Kopie der erstellten Protokolldatei.
