KOPIEREN EINES HRE-PROJEKTS

Dieser Bildschirm ermöglicht es Ihnen, ein HRE-Projekt von einer beliebigen Quelle zu einem beliebigen Ziel zu kopieren. Es kann dann wie gewünscht geöffnet werden. Der Standard-Ort für Kopien kann unter Werkzeuge > Einstellungen > Benutzer > Datei-Speicherorte festgelegt werden.

1. Sie können ein Projekt aus der Liste "Bekannte Projekte" auswählen oder mit DURCHSUCHEN das Fenster "Dateiauswahl" öffnen, um eine HRE-Datei auf diesem Computer, auf einer entfernten Quelle oder einem Wechseldatenträger zu finden. Wählen Sie das gewünschte Projekt.

2. In KOPIEREN NACH geben Sie dem neuen Projekt einen Namen

3. Verwenden Sie erneut die Dateiauswahl, um nach dem Zielordner des Projekts zu DURCHSUCHEN und vergeben Sie den neuen Dateinamen

4. Drücken Sie KOPIEREN, um den Vorgang zu starten

5. Dieses Projekt kann nun auf diesem oder einem anderen Computer geöffnet werden.

Verwandte Themen

  • Projekt Öffnen
  • Importieren eines Projekts von TMG
  • Projekt-Zusammenfassung
  • Projekt Sichern
  • Projekt Wiederherstellen
  • Projekt Schließen
  • Projekt Umbenennen
  • Projekt Löschen
