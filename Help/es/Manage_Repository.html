Administrar repositorio

La pantalla Administrar repositorio le permite ver, editar, copiar, agregar y eliminar datos del repositorio.

El uso de las funciones 'Agregar' y 'Editar' muestra la pantalla de detalles del Repositorio, que permite configurar el nombre del Repositorio, el nombre abreviado, la ubicación (de acuerdo con el Estilo de ubicación seleccionado) y una Nota.

Un repositorio que está en uso no puede eliminarse.

<

Los repositorios se pueden encontrar en la lista ingresando texto en el cuadro de texto 'Buscar'.
