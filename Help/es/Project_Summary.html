Resumen del proyecto HRE

Esta pantalla permite al usuario examinar un resumen de un Proyecto HRE actual.

1. El resumen del proyecto es una breve descripción general de un proyecto HRE seleccionado

2. Se puede ver como una opción directa desde el menú PROYECTO, o se puede ver para un proyecto abierto elegido desde varios de los menús de acción: Abrir proyecto, Copia de seguridad del proyecto, Cerrar proyecto, Copiar proyecto (breve resumen), Renombrar proyecto, Eliminar proyecto y Salir de HRE (Cerrar procedimiento)

3. El resumen muestra:

  • la hora actual
  • el nombre del proyecto seleccionado por el creador
  • el nombre de archivo utilizado para almacenar los datos del proyecto
  • el nombre de la carpeta donde se almacena
  • el servidor/nombre de la computadora
  • cuándo se cerró el proyecto por última vez
  • cuándo se realizó la última copia de seguridad del proyecto
  • el tipo y la versión de la base de datos HRE.
