Cerrar un proyecto HRE abierto

Esta opción le permite Cerrar un proyecto que está actualmente abierto en HRE.

1. Para cerrar cualquier proyecto abierto actualmente, seleccione PROYECTO en el menú y luego seleccione la opción CERRAR PROYECTO

2. Aparece la lista de proyectos abiertos, que muestra el nombre del proyecto, el nombre del archivo y si es un proyecto local o uno en un servidor

3. Resalte el nombre del Proyecto que desea cerrar. En la ilustración, el Proyecto 'Mi Taller' ha sido seleccionado de los proyectos Abiertos actualmente. Si solo hay un proyecto abierto, se selecciona de forma predeterminada

4. Una vez que resalte el Proyecto requerido, el nombre aparece debajo, para habilitar la verificación

5. Tenga en cuenta que en este punto, el Proyecto seleccionado también puede estar sujeto a otros procesos, incluidos SUMMARY, VALIDATE o BACKUP, y hay una opción para CANCELAR el cierre

6. Si luego selecciona CERRAR PROYECTO y confirma la decisión, el Proyecto se cerrará (no habrá ningún aviso de confirmación si se seleccionó esa opción en AJUSTES). HRE permanece abierto para que se pueda abrir otro Proyecto si se desea

7. Si el Proyecto cerrado era actualmente el Proyecto activo, puede optar por convertir otro Proyecto en Activo.
