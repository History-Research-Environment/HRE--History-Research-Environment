Administración de proyecton

Esta pantalla permite al administrador de un proyecto definir los usuarios que pueden tener acceso a un proyecto HRE.
De forma predeterminada, el creador original de un proyecto HRE tiene todos los derechos de administración del proyecto y luego puede usar esta función para agregar otros usuarios.

La pantalla muestra una tabla de proyectos conocidos actualmente para esta implementación de HRE, seguida de una tabla de los usuarios conocidos, que se completa cuando se selecciona un proyecto en la tabla de proyectos.
También se puede buscar un proyecto mediante el botón 'Examinar...'.

Se proporcionan controles para agregar y eliminar usuarios y cambiar la contraseña de un usuario.
Las contraseñas deben tener al menos 8 caracteres, incluida una letra mayúscula, una letra minúscula, un número y un carácter especial.
