Copia de seguridad de un proyecto HRE

Esta pantalla permite te hacer una copia de seguridad de un proyecto HRE existente, ya sea que el proyecto esté abierto actualmente en HRE o no. Te puede seleccionar el destino del archivo de copia de seguridad y elegir si desea o no hacer una copia de seguridad de las carpetas externas que forman parte del proyecto. La ubicación predeterminada de la copia de seguridad se puede configurar en Herramientas > Configuración > Usuario > Ubicaciones de archivos.

1. Seleccione PROYECTO >> PROYECTO DE RESPALDO y resalte el Proyecto deseado de la lista presentada

2. HRE ahora mostrará el nombre de archivo que se aplicará a la copia de seguridad, que es el nombre del proyecto con la hora y la fecha actuales agregadas. En este punto hay una opción para ver un RESUMEN del Proyecto seleccionado

3. Haga clic en EXAMINAR para ubicar la carpeta donde se almacenará la copia de seguridad. Aunque aparecerá la carpeta predeterminada (como se define en la configuración de HRE para ubicaciones de archivos), puede navegar para seleccionar cualquier otra ubicación. El archivo recibirá automáticamente el sufijo de archivo de copia de seguridad compactado .hrez. Una vez que la carpeta de ubicación deseada esté en la ventana, presione ENTER para seleccionarla

4. Ahora puede optar por hacer una copia de seguridad de las carpetas de archivos externas (imágenes, documentos, archivos de sonido, videos, etc.) que están asociadas con este proyecto marcando la casilla de selección. Esto abrirá una nueva ventana que le permitirá seleccionar una o más carpetas para hacer una copia de seguridad y especificar dónde desea guardarlas, que puede ser diferente de la ubicación de la copia de seguridad del proyecto. El nombre del archivo de copia de seguridad contendrá el nombre de la carpeta original y será un tipo de archivo zip estándar.

5. Ahora haga clic en RESPALDO para iniciar el proceso. Si el proyecto nominado está actualmente abierto en HRE, primero se cerrará y luego comenzará la copia de seguridad. Primero se realizará una copia de seguridad de la base de datos del proyecto, seguida de Carpetas de archivos externas (si especificó alguna).

6. Alternativamente, seleccione CANCELAR si la copia de seguridad ya no es necesaria.
