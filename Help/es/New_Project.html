CREAR UN NUEVO PROYECTO

Esta pantalla le permite crear un nuevo proyecto, definir su nombre y nombre de archivo, y designar la ubicación donde se deben almacenar los archivos de la base de datos.

Tenga en cuenta que esto NO debe usarse para importar un conjunto de datos TMG. El proceso para Importar datos desde TMG se selecciona por separado desde el menú HERRAMIENTAS.

Seleccionar Crear proyecto en el menú Proyecto proporciona un proceso de tres pasos a seguir:

1. Asigne un nombre a su proyecto para identificarlo en el futuro. Este debe ser descriptivo y puede constar de varias palabras, pero debe ser único

2. Haga clic en EXAMINAR para elegir dónde se almacenará la base de datos. Se presentará una ubicación predeterminada, pero puede elegir otra si es necesario

3. Una vez que se haya seleccionado la ubicación del archivo, asigne a su Proyecto un nombre de archivo conciso, que debe ser diferente a cualquier otro archivo de Proyecto HRE almacenado en esa ubicación. Se deben seguir las convenciones de Windows/MAC en cuanto a los caracteres que puede usar en los nombres de archivo y HRE agregará automáticamente el sufijo de archivo .mv.db. Pulse SELECCIONAR.

Ahora la ventana estará completa mostrando su nuevo nombre de proyecto, nombre de archivo y ubicación

En este punto, puede optar por CANCELAR o CREAR el nuevo Proyecto.

Si ya ha creado el proyecto, se inicializará, abrirá y ACTIVARÁ. Aparecerá una ventana para indicar que el proceso se ha completado.

Ahora puede comenzar a ingresar datos en el nuevo Proyecto, u optar por cambiar a otro proyecto abierto para estar Activo, o Abrir otro proyecto.
