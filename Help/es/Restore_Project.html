Restaurar un proyecto HRE respaldado

Esta pantalla permite te restaurar copias de seguridad anteriores y luego trabajar con ese archivo como el Proyecto abierto actual.
El proyecto original, si aún está presente, solo se sobrescribirá si te lo permite específicamente.

1. Haga clic en ENCONTRAR UN ARCHIVO DE RESPALDO y desde el Selector de archivos seleccione el archivo que desea restaurar; como alternativa, busque cualquier otra ubicación y SELECCIONE el archivo

2. Su selección, incluido su nombre descriptivo, se transferirá nuevamente a la ventana Restauración del proyecto. Consulte las NOTAS sobre las opciones desde aquí

3. Establezca la ubicación de la carpeta donde se restaurará la copia de seguridad, nuevamente desde la ventana del Selector de archivos

4. El nombre del archivo es el utilizado por defecto en el proyecto original, pero DEBE cambiarse si se va a utilizar el mismo destino para no sobrescribir el archivo original

5. Es fundamental que al proyecto restaurado se le dé un nuevo nombre descriptivo para diferenciarlo del proyecto original. Una vez ingresado esto, haga clic en RESTAURAR

6. Cuando se complete, el proyecto restaurado será el Proyecto Activo

7. Al verificar, si el proyecto restaurado es lo que se requería, luego puede optar por eliminar la versión anterior del proyecto si todavía está en esta computadora.

8. La segunda parte de la pantalla permite la restauración de archivos externos (imágenes, documentos, etc.) desde sus archivos zip de copia de seguridad creados en Project Backup. Tenga en cuenta que las dos partes de la pantalla funcionan de forma independiente; en otras palabras, puede restaurar un archivo externo sin restaurar primero un archivo de proyecto.
