CAMBIAR EL NOMBRE DE UN PROYECTO

Esta opción del Menú PROYECTO le permite cambiar el nombre de un Proyecto HRE existente, conservando la base de datos en su ubicación actual.

1. Resalte el proyecto al que desea cambiar el nombre. En este ejemplo, el Proyecto se llama "Inmigrantes". Solo se puede cambiar el nombre de los proyectos a los que se accedió anteriormente

2. Tenga en cuenta que no puede cambiar el nombre de un proyecto actualmente abierto (aparecerá un mensaje emergente de advertencia); primero debe cerrarse

3. Para fines de verificación, se puede ver un RESUMEN del proyecto elegido antes de continuar

4. En el espacio provisto, ingrese el nuevo nombre para este Proyecto. Se permiten espacios, números, mayúsculas y minúsculas y los caracteres habituales de los nombres de archivo. El nuevo nombre debe ser un nombre de proyecto único, independientemente de su ubicación

5. Haga clic para RENOMBRAR el proyecto o CANCELAR para no continuar

6. Ahora se ha cambiado el nombre del Proyecto, a "Nuevos inmigrantes" en este ejemplo, y la lista de Proyectos se actualizará

7. El proceso de cambio de nombre del Proyecto es solo para fines de acceso e identificación del usuario, ya que el Nombre de archivo ("Immigrants_to_AUS" en este caso) permanece sin cambios en la computadora.
