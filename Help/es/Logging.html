Informe de registro de HRE

Esta pantalla permite al usuario examinar los archivos de registro de actividad de HRE que se crean cada vez que se abre HRE y optar por realizar varias tareas en esos archivos.

1. Seleccione INFORMES >> REGISTRO

2. Haga clic en EXAMINAR para abrir la carpeta predeterminada para los archivos de registro de HRE, como se muestra

3. Ahora está a la vista una lista de los archivos de registro diarios, y al hacer doble clic en cualquiera de ellos se abrirá el archivo, que muestra toda la actividad en HRE durante ese día en particular

4. El archivo de registro puede contener una gran cantidad de datos bajo los encabezados: fecha, hora, servidor (computadora), nombre del proyecto y entrada de registro. Tenga en cuenta que la última columna puede incluir acciones del programa y errores

5. Ahora puede enviar el contenido del registro, o una parte seleccionada del mismo, a un archivo o impresora. Esto puede ser valioso para el soporte técnico o para volver sobre los pasos realizados en una sesión de HRE. Para Salir todo o una Selección, use el mouse para resaltar las líneas elegidas del registro. El destino puede ser un archivo o una impresora, incluido un archivo PDF. Una vez hecho esto, hacer clic en Restablecer selección restaura todo el archivo de registro para verlo una vez más, o permite elegir otro archivo de registro.

6. Cierre la ventana de registro cuando haya terminado.
