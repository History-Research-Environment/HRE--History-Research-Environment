Configuración HRE

Esta pantalla le permite establecer una serie de preferencias sobre cómo funciona HRE. La configuración se puede cambiar en cualquier momento y no se ve afectada por el cierre y la reapertura de HRE o sus Proyectos.

1. Desde el menú principal HERRAMIENTAS, seleccione CONFIGURACIÓN, luego USUARIO para seleccionar y configurar un rango de opciones de Usuario.

2. Desde el conjunto de pestañas, puede elegir cambiar:

    HRE Look and Feel: para establecer el 'look and feel': el tema / tamaño y tipo de fuente (el predeterminado de HRE es Graphite, Blue-medium font, Arial)

    Configuración de idioma: para configurar los idiomas predeterminados para las pantallas, datos e informes de HRE

    Configuración de formato: para configurar los formatos de fecha y hora

    Opciones de usuario:

  • Mostrar la pantalla de bienvenida al iniciar HRE (predeterminado: No)
  • Mostrar solicitudes de confirmación para cancelar y otras acciones (predeterminado: Sí)
  • Abrir el último proyecto de HRE cerrado al iniciar HRE (predeterminado: No)
  • Hacer una copia de seguridad de un proyecto al cerrarlo (predeterminado: No)
  • Habilitar el procesamiento del código de complemento (predeterminado: No)
  • Recargar siempre el selector de personas al agregar una persona a la base de datos (predeterminado: Sí). Si esta opción está deshabilitada, el selector de personas solo se recarga cuando se solicita desde el menú de HRE
  • Solicitar siempre el apellido de casada al guardar matrimonios (predeterminado: No)

    Acentos: para establecer los colores de la fuente y el fondo para cualquier tema de HRE que desea definir

    Ubicaciones de archivos: para configurar las ubicaciones predeterminadas para proyectos, informes, copias de seguridad y copias de seguridad externas

    Opciones de depuración: para configurar la escritura de un archivo de registro (predeterminado), la escritura de mensajes de depuración y mensajes de tiempo. Estas opciones son principalmente para su uso por soporte técnico HRE

    Modo del Servidor: Para habilitar la ejecución de HRE en el modo Servidor (el valor predeterminado es No)
