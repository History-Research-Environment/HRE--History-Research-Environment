Definición de Proyecto Remoto

El cuadro de diálogo Definición de Proyecto Remoto permite definir un Proyecto HRE existente en un servidor remoto o editar dicha definición, según la selección de una de estas funciones en la pantalla inicial.

Seleccionar "Editar un Proyecto Remoto" permite elegir un proyecto remoto de la lista de todos los proyectos de este tipo que se conocen actualmente en la instalación de HRE.
Una vez seleccionado un proyecto remoto, la pantalla se expande para mostrar dos secciones de información del proyecto para su edición.
Seleccionar "Definir un Nuevo Proyecto Remoto" muestra las mismas dos secciones de información del proyecto, en este caso para la entrada inicial.

El primer conjunto de información es la definición del Proyecto Remoto: su Nombre, Nombre de Archivo y Ubicación en el servidor remoto.
El segundo conjunto de información define el Remoto Servidor: su nombre, dirección IP y número de puerto de acceso.

Toda esta información le será proporcionada por el administrador del servidor remoto, quien habrá creado el proyecto HRE al que desea acceder.

Una vez completada la información, puede guardarla y podrá acceder al proyecto HRE remoto definido mediante la pantalla "Abrir proyecto" de HRE.
