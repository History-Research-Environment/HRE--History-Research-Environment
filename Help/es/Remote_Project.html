DEFINICIÓN DE PROYECTO REMOTO

El cuadro de diálogo Definición de proyecto remoto le permite definir un proyecto HRE que existe en un servidor remoto, o editar dicha definición, según su selección de una de estas dos funciones en la pantalla inicial.

La selección de 'Editar un proyecto remoto' le permite elegir un proyecto remoto de la lista de todos los proyectos conocidos actualmente en su instalación de HRE.
Una vez que se selecciona un proyecto remoto, la pantalla se expande para mostrar 2 secciones de información del proyecto para editar.
Al elegir 'Definir un nuevo proyecto remoto' se muestran las mismas 2 secciones de información del proyecto, en este caso para la entrada inicial.

El primer conjunto de información es la definición del proyecto remoto: su nombre, nombre de archivo y ubicación en el servidor remoto.
El segundo conjunto de información define el servidor remoto: su nombre, dirección IP y número de puerto de acceso.

Toda esta información te la proporcionará el Administrador del servidor remoto, quien habrá creado el Proyecto HRE al que deseas acceder.

Cuando la información esté completa podrás guardarla. El acceso al proyecto HRE remoto definido será posible mediante el uso de la pantalla normal 'Abrir proyecto' de HRE.
