Administrar ubicación

La pantalla "Administrar ubicación" le permite editar los distintos elementos que definen los datos de una ubicación, según el estilo de ubicación seleccionado.

También puede establecer un rango de fechas para definir el período durante el cual esta ubicación se conoció con este nombre.

Seleccionar la sección "Medios" le permite agregar medios relevantes para describir esta ubicación.
