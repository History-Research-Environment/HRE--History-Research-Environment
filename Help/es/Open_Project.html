ABRIR UN PROYECTO HRE

Esta pantalla le permite abrir un proyecto HRE existente desde cualquier ubicación elegida. Entonces se convertirá en el PROYECTO ACTIVO.

HRE permite que hasta TRES proyectos estén abiertos en cualquier momento, pero solo uno puede ser nominado para estar ACTIVO. Este será el último proyecto abierto O el proyecto seleccionado mediante el uso del símbolo de asesoramiento de proyecto "P" al final de la barra de menú.

APERTURA RÁPIDA: los proyectos HRE usados ​​recientemente se muestran como una lista en la parte inferior del menú Proyecto, debajo de las opciones del menú. Los proyectos ​​recientes, identificados con el logotipo de HRE, se pueden abrir haciendo clic en el nombre del proyecto.


1. Para abrir cualquier proyecto HRE existente, BUSQUE hasta la ubicación de la base de datos si no se muestra en la lista predeterminada. Los proyectos pueden ser locales, en una red o en una unidad extraíble

2. Se mostrarán el nombre del proyecto seleccionado y el nombre del archivo actual, y se podrá ver un RESUMEN

3. Haga clic en ABRIR (o haga doble clic en el nombre del Proyecto) para abrir el Proyecto, o CANCELAR para salir de esta pantalla

4. Una vez que se ha abierto un Proyecto, automáticamente se convertirá en el Proyecto Activo

5. En cualquier momento, puede optar por cambiar el Proyecto Activo.
