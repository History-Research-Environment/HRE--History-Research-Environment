Editar tipos de fuentes

La pantalla "Editar tipos de fuente" permite ver y editar los datos del tipo de fuente seleccionado.

Puede cambiar el nombre del tipo de fuente y editar cualquiera de las tres plantillas para notas al pie y entradas de bibliografía, además del texto recordatorio.

Recibirá mensajes de error si intenta guardar nombres de elementos de fuente desconocidos o utiliza códigos de formato no válidos, como [ITAL].
