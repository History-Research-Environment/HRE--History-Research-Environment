SALIR de HRE

Esta opción le permite cerrar Proyecto(s) Abierto(s) y salir del programa HRE, con la opción de realizar otros procesos antes del cierre.

Hay dos formas de salir de HRE y hay ligeras variaciones, pero el resultado final es el mismo.

A] DESDE EL MENÚ DE PROYECTOS

1. Seleccione Cerrar proyecto en el menú principal del proyecto

2. Haga clic en el proyecto que desea cerrar, para que se resalte

3. Ahora tienes cinco opciones
- Para ver un Resumen del Proyecto seleccionado
- Para Validar el Proyecto, asegurando que los índices y otras partes sean correctas y completas, y corrigiendo automáticamente cualquier error si es posible
- Para hacer una copia de seguridad del proyecto antes de cerrarlo, dándole la opción de grabar su copia de seguridad en una ubicación elegida
- Para Cancelar el cierre
- Para realmente cerrar el proyecto.

4. A medida que se cierra cada proyecto, un mensaje informará que esto ha sucedido, a menos que se hayan desactivado las indicaciones de cancelación en la configuración de HRE

5. HRE cierra cuando se han cerrado todos los proyectos abiertos

6. Tenga en cuenta que el último proyecto cerrado será el proyecto que se abra automáticamente en el próximo uso de HRE si se ha activado la configuración de HRE para 'Abrir el último proyecto cerrado'.


B] DESDE LA X EN LA PARTE SUPERIOR DERECHA DE LA PANTALLA PRINCIPAL DE HRE

1. Se le pedirá que confirme que desea salir de HRE, a menos que las indicaciones de cancelación se hayan desactivado en la configuración de HRE

2. Si solo hay un proyecto abierto, se cerrará

3. Si hay más de un proyecto abierto, se abrirá la pantalla Cerrar proyecto descrita anteriormente

4. A medida que se cierra cada proyecto, un mensaje informará que esto ha sucedido, a menos que se hayan desactivado las indicaciones de cancelación en la configuración de HRE

5. HRE cierra cuando se han cerrado todos los proyectos abiertos

6. Tenga en cuenta que el último proyecto cerrado será el proyecto que se abra automáticamente en el próximo uso de HRE si se ha activado la configuración de HRE para 'Abrir el último proyecto cerrado'.
