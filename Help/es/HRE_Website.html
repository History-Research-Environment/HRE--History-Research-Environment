TEL SITIO WEB HRE

Al seleccionar la opción Sitio web de HRE, se abre su navegador predeterminado y se muestra la página de entrada del sitio web de HRE.

Este sitio web contiene el último boletín de HRE (y sus ediciones históricas), permite que los voluntarios se registren para ayudar en el desarrollo de HRE y contiene otros artículos de interés para los usuarios de HRE.
