Administrar fuentes

La pantalla Administrar fuentes muestra una lista de las fuentes definidas y cuántas veces han sido citadas.
La pantalla le permite ver, editar, copiar, agregar y eliminar datos de fuentes, pero una fuente que esté en uso no se puede eliminar.

Cuando se selecciona una fuente, también se muestran las plantillas para sus estructuras de notas al pie y bibliografía.
Para cambiar estas plantillas, utilice el botón 'Editar' (o haga doble clic en la entrada seleccionada).

El uso del botón 'Agregar' permite la creación de una nueva fuente y 'Editar' permite la manipulación de la definición de una fuente actual.

El botón 'Mostrar inactivos' muestra solo las fuentes que se han marcado como inactivas, pero que aún pueden estar presentes en la base de datos.

Las fuentes se pueden encontrar en la lista ingresando texto en el cuadro de texto 'Buscar'.
