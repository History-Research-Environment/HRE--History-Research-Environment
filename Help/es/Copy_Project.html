COPIAR UN PROYECTO HRE

Esta pantalla permite te copiar un Proyecto HRE desde cualquier fuente a cualquier destino elegido. Luego se puede abrir según sea necesario. La ubicación de copia predeterminada se puede configurar en Herramientas > Configuración > Usuario > Ubicaciones de archivos.

1. Puede seleccionar un proyecto de la lista de Proyectos conocidos o usar EXAMINAR para abrir la ventana del Selector de archivos para encontrar un archivo HRE en esta computadora, en una fuente remota o en una unidad extraíble. Seleccione el Proyecto requerido

2. En COPY TO dale un nombre al nuevo proyecto

3. Vuelva a utilizar el Selector de archivos para EXAMINAR la carpeta de destino del proyecto y asigne el nuevo nombre de archivo

4. Pulse COPIAR para que se lleve a cabo el proceso

5. Este proyecto ahora se puede abrir en esta computadora o en otra computadora.
