IMPORTAR UN PROYECTO TMG A HRE

HRE importará directamente un proyecto desde la base de datos The Master Genealogist (TMG) versión 8.05 o 9.x

PREPARANDO TMG:

Se deben realizar una serie de procesos en TMG para ayudar a realizar una importación exitosa. Estos son:

1. Banderas: en el menú Archivo TMG, haga clic en la entrada Administrador de banderas. Cuando se abra, simplemente cierre la pantalla. Esta simple acción reparará una serie de inconsistencias que pueden estar presentes en las entradas de la bandera TMG.

2. Miniaturas de exhibición: bajo ciertas condiciones, las miniaturas de exhibición de TMG se pueden perder o dañar y no se pueden importar a HRE. Para solucionar este problema, realice lo siguiente: abra cualquier registro de exhibición de TMG que tenga una entrada y haga clic derecho en la exhibición. Seleccione 'Actualizar todas las miniaturas'. Tenga en cuenta que esto puede tardar algún tiempo en finalizar si tiene una gran cantidad de exhibiciones.


ANTES DE INICIAR UNA IMPORTACIÓN:

Verifique que el proyecto TMG solo contenga un conjunto de datos: el proceso de importación solo importará el primer conjunto de datos de un proyecto TMG.
Verifique la ubicación de su archivo de proyecto TMG (.pjc), como se especifica en TMG: Archivos >> Preferencias >> Opciones del proyecto actual >> General, para que pueda encontrarlo rápidamente una vez que comience el proceso de importación.
Decida dónde va a almacenar la base de datos HRE cuando la importe. Si no necesariamente desea utilizar la ubicación predeterminada de HRE, cree una carpeta (por ejemplo, Proyectos HRE) en la ubicación deseada y en HRE >> Herramientas >> Configuración >> Usuario >> Ubicaciones de archivos puede especificar esta carpeta como su por defecto.


EL PROCESO DE IMPORTACIÓN:

1. Seleccione Herramientas, luego Importar y aparecerá la opción 'Desde TMG'. Haga clic en este elemento del menú

2. Ingrese el NOMBRE del proyecto que desea usar para este proyecto en HRE

3. Haga clic en el segundo cuadro para seleccionar la UBICACIÓN donde desea guardar el nuevo proyecto. De forma predeterminada, se utilizará la configuración actual de Ubicaciones de archivos HRE, pero puede cambiar esta ubicación si es necesario, utilizando la administración normal de archivos y carpetas. Ingrese el nombre de archivo que desea usar para este proyecto

4. Luego seleccione la ubicación de su base de datos TMG, haciendo clic en su archivo .pjc

5. Una vez nominado, comenzará el proceso de importación

6. Aparecerá una ventana de progreso a medida que HRE recorre la base de datos TMG elegida y la lee en HRE como un nuevo proyecto HRE

7. Al hacer clic en Cerrar, los datos de TMG estarán disponibles para que los utilice HRE

8. Utilice el elemento de menú HRE Open Project para abrir el proyecto

9. Este puede ser un momento oportuno para realizar una copia de seguridad de este nuevo proyecto HRE, antes de realizar cualquier edición.

NOTA: La base de datos de TMG no se modifica de ninguna manera. Permanece en su carpeta original y seguirá estando disponible en TMG.


ERRORES EN EL PROCESO DE IMPORTACIÓN:

Si el proceso de importación falla debido a errores en la base de datos de TMG que HRE no puede manejar, el soporte de HRE puede solicitarle que proporcione más información de depuración mediante el siguiente proceso.
Antes de continuar, elimine la base de datos HRE probablemente incompleta que acaba de crear utilizando el elemento de menú "Eliminar proyecto" en el elemento de menú Proyecto de HRE. Cuando se le pregunte si desea conservar la base de datos, responda "No".

Ahora vaya a HRE >> Herramientas >> Configuración >> Usuario >> Opciones de depuración y haga clic en 'Escribir mensajes de DEPURACIÓN', luego acepte el cambio.

Vaya al Paso 1 del PROCESO DE IMPORTACIÓN DE TMG anterior. La pantalla de importación habrá cambiado y ahora mostrará 6 casillas de verificación y la ubicación donde se creará un nuevo archivo de registro.

Seleccione las casillas de verificación que le haya indicado el soporte de HRE y repita los pasos del proceso de importación de TMG. Cuando el proceso de importación vuelva a fallar, cierre HRE y envíe por correo electrónico al soporte técnico una copia del archivo de registro que se habrá creado.
