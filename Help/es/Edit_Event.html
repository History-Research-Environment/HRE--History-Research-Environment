Editar Evento

La pantalla Editar Evento le permite editar el Evento seleccionado de su proyecto.
El Idioma predeterminado es el idioma actual de los Datos HRE, pero puede cambiarse para editar el Evento en cualquier otro idioma compatible.

La pantalla consta de 3 pestañas: Configuración General, Roles y Sentencias, además de Sugerencias de Evento.

Configuración General:
Aquí puede editar el nombre del Evento y el grupo de eventos al que pertenece.
La configuración de GEDCOM define cómo debe exportarse a GEDCOM: ya sea con su propio nombre de etiqueta o como una etiqueta de Evento con un subtipo igual al nombre.
La configuración de Validez del Evento define en qué rango de años calendario se puede usar el evento y el número mínimo de Asociados Principales requerido para este evento.

Roles y Oraciones:
Esta sección permite editar los roles que se pueden aplicar a los asociados del evento y si son roles primarios (obligatorios).
Las estructuras de las oraciones se pueden editar para cada rol y cada rol se puede restringir por sexo y edad.
Ten en cuenta que al cambiar el idioma del evento, los roles se ajustarán a los aplicables a ese idioma.

Sugerencias del evento:
Esta pestaña ofrece un área de texto libre para que describas cualquier sugerencia que desees registrar para facilitar el uso del evento.
