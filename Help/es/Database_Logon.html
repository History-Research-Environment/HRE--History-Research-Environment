Inicio de sesión en la base de datos HRE

Este cuadro de diálogo aparece cuando intenta abrir un proyecto HRE definido en un servidor remoto.

La pantalla muestra 3 áreas de entrada de datos, como sigue:
1. Datos de conexión remota, consistentes en el tipo Base de Datos; dirección IP del servidor; número de puerto del servidor. Esta información es principalmente para fines de confirmación y normalmente no se espera que usted la cambie

2. Su ID de inicio de sesión para este proyecto, proporcionado por el administrador del sistema del servidor remoto y guardado por HRE

3. Su contraseña para este proyecto, tal como la proporcionó inicialmente el administrador del sistema del servidor remoto

NOTA:

* HRE solo permitirá 3 intentos de inicio de sesión antes de fallar el proceso de inicio de sesión.

* El uso de un servidor remoto requiere que HRE esté en 'modo servidor'; esto se configura en Herramientas->Configuración->Modo Servidor.
