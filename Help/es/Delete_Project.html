ELIMINAR UN PROYECTO

Esta pantalla permite al usuario eliminar un Proyecto sobrante.

Tenga en cuenta que el proyecto no puede estar abierto actualmente en HRE; debe cerrarse primero.

1. Seleccione el proyecto que desea Eliminar. El nombre del proyecto, el nombre del archivo del proyecto y la ubicación se completarán debajo del cuadro de selección

2. Sin duda, consultar el RESUMEN del proyecto puede ser una buena idea en este momento

3. Si continúa, presione ELIMINAR

4. Conservar base de datos: se le preguntará si desea conservar el archivo de la base de datos (mientras elimina la referencia del proyecto HRE) o eliminar ambos. Responda NO para eliminar todo o SÍ para conservar el archivo de la base de datos.
