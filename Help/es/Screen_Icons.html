ICONOS DE PANTALLA HRE

Estos íconos brindan acceso rápido a algunas de las funciones e instalaciones de HRE.

Ayuda: Haga clic para mostrar las páginas de ayuda apropiadas para la pantalla

Salida: Haga clic para generar cualquier tabla de pantalla en una copia impresa o en un archivo csv/tsv

Recordatorio: Haga clic para abrir el cuadro de diálogo Recordatorio, donde puede crear notas de texto sobre el uso de la pantalla

Configurar: Haga clic para configurar las tablas que se muestran en pantalla

Copiar persona: Haga clic para copiar la persona que se muestra actualmente

Eliminar persona: Haga clic para eliminar la persona que se muestra actualmente

Renumerar persona: Haga clic para volver a numerar la persona que se muestra actualmente
