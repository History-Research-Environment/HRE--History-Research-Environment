Administrar Elementos Fuente

La pantalla "Administrar Elementos Fuente" permite manipular los Elementos Fuente en todos los idiomas compatibles.
Inicialmente, muestra los elementos del idioma de datos HRE configurado.

Al hacer doble clic en un elemento seleccionado, se puede editar su nombre.

El botón "Agregar" permite crear nuevos Elementos Fuente en el idioma seleccionado.

Tenga en cuenta que si agrega un elemento en el idioma seleccionado, no habrá ningún elemento correspondiente en ningún otro idioma.
Esto se puede corregir con el botón "Traducir", que permite agregar un elemento equivalente en otro idioma.

Los elementos se pueden encontrar en la lista de elementos introduciendo texto en el cuadro de texto "Buscar".
