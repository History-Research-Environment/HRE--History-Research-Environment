Modifier le dépôt

L'écran « Modifier le référentiel » vous permet de consulter et de modifier les données du référentiel sélectionné.

Ces données incluent le nom du référentiel, son nom abrégé, son emplacement (selon le style d'emplacement sélectionné) et une note.
