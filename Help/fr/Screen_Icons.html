Icônes d'écran HRE

Ces icônes permettent d'accéder rapidement à certaines fonctionnalités et installations de HRE.

Aide : Cliquez pour afficher les pages d'aide appropriées à l'écran

Afficher : Cliquez pour afficher n'importe quel tableau d'écran sur une impression ou un fichier csv/tsv.

Rappel : Cliquez pour ouvrir la boîte de dialogue Rappel, dans laquelle vous pouvez créer des notes textuelles lors de l'utilisation de l'écran.

Configurer : Cliquez pour configurer les tableaux affichés à l'écran

Copier la personne : Cliquez pour copier la personne actuellement affichée

Supprimer la personne : Cliquez pour supprimer la personne actuellement affichée

Renuméroter la personne : Cliquez pour renuméroter la personne actuellement affichée
