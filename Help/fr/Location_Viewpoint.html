POINT DE VUE DE L'EMPLACEMENT

Le Point de vue de localisation permet à l'utilisateur de voir toutes les informations enregistrées dans HRE pour l'emplacement sélectionné. Jusqu'à 5 points de vue de localisation peuvent être ouverts pour chaque projet HRE ouvert
L'écran s'ouvre initialement dans un format réduit affichant uniquement le nom de l'emplacement et 6 barres de titre de sous-écran.

Les 6 sous-écrans définis par les barres de titre se trouvent à gauche et à droite d'une barre de séparation verticale. Le séparateur peut être déplacé vers la gauche ou la droite pour modifier la quantité d'espace d'écran alloué à chaque moitié de l'écran. En haut du séparateur se trouvent deux petits boutons fléchés qui peuvent être utilisés pour que l'un ou l'autre côté occupe tout l'écran.

Chacun des sous-écrans peut être affiché en cliquant sur le triangle rouge à gauche de chaque barre de titre, ce qui change le triangle en un format vert pointant vers le bas et ouvre le sous-écran. Cliquer sur un triangle vert réduit le sous-écran. Chaque barre de titre indique également le nombre d'entrées contenues dans le sous-écran, par exemple « Événements (6) ».

Lorsqu'un sous-écran est affiché, il peut également afficher les boutons "+" et "-" en haut à droite. Ceux-ci permettent d'augmenter ou de réduire le nombre de lignes de données affichées dans le sous-écran..

Si la hauteur de l’écran Point de vue devient plus grande que l’écran HRE qui l’entoure, l’écran Point de vue passera à un format déroulant.

Sous-écran Événements :
Chaque entrée affiche des informations sur un événement se produisant à cet emplacement. Une case à cocher permet de restreindre la liste pour afficher uniquement les événements pour lesquels il y avait un Associé Clé.

Sous-écran Supérieur/Inférieur :
Ceci n’est pas encore mis en œuvre dans HRE.

Sous-écran Associés :
Cela affiche une liste de toutes les personnes associées à l'emplacement sélectionné via leur implication dans des événements survenus à cet emplacement.

Sous-écran Images :
Affiche toutes les images (le cas échéant) de l'emplacement.

Sous-écran Indicateurs :
Affiche tous les indicateurs Actifs pour l'emplacement et leurs paramètres actuels.

Sous-écran Bloc-notes :
Les blocs-notes ne sont pas encore implémentés dans HRE.
