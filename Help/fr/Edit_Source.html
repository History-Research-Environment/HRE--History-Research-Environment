Ajouter et modifier des sources

L'écran d'ajout/modification des sources vous permet de créer ou de modifier toutes les données de source.

Cet écran comporte trois onglets : Données générales de la source, Informations complémentaires et Connexions de cette source aux sources et aux référentiels.

Onglet Général :
Cet onglet définit le titre de la source et son titre abrégé (les deux doivent être présents) ainsi que le type de source dont ils sont issus.
Selon le type de source sélectionné, il affiche également les modèles de notes de bas de page et de bibliographie applicables, ainsi que tous les éléments de source pertinents associés à ces modèles, avec leurs valeurs (le cas échéant).
Si un modèle a été modifié pour s'appliquer uniquement à cette source (un « modèle local »), il est identifié comme ayant remplacé le modèle global (type de source). Modèle.
Les boutons « Aperçu » permettent de visualiser le résultat de l'insertion des valeurs actuelles des éléments sources dans chacun des modèles.

Onglet supplémentaire :
Cet onglet permet de définir la fidélité de la source et son état (actif/inactif).
D'autres données peuvent être enregistrées, notamment la référence et les rappels de la source.
Si une personne définie dans le projet est également auteur, éditeur ou compilateur de ces données sources, elle peut également être enregistrée ici.

Onglet Connexions :
Permet d'ajouter (ou de supprimer) toute source de cette source et de la lier à un référentiel.
