DÉFINITION DE PROJET À DISTANCE

La boîte de dialogue Définition de projet distant vous permet de définir un projet HRE qui existe sur un serveur distant, ou de modifier une telle définition, en fonction de votre sélection de l'une de ces deux fonctions dans l'affichage initial.

La sélection de « Modifier un projet distant » vous permet de choisir un projet distant dans la liste de tous ces projets actuellement connus de votre installation HRE.
Une fois qu'un projet distant est sélectionné, l'écran s'agrandit pour afficher 2 sections d'informations sur le projet à modifier.
Choisir « Définir un nouveau projet distant » affiche les deux mêmes sections d'informations sur le projet, dans ce cas pour la saisie initiale.

Le premier ensemble d'informations est la définition du projet distant : Nom, Nom de fichier et Emplacement sur le serveur distant.
Le deuxième ensemble d'informations définit le serveur distant : Nom, Adresse IP et Numéro de port d'accès..

Toutes ces informations vous seront fournies par l'Administrateur du serveur distant, qui aura créé le Projet HRE auquel vous souhaitez accéder.

Lorsque les informations sont complètes, vous pouvez les enregistrer. L'accès au projet HRE distant défini sera alors possible en utilisant l'écran normal « Ouvrir le projet » de HRE.
