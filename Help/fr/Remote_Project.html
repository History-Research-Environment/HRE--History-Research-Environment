Définition d'un projet distant

La boîte de dialogue Définition d'un projet distant vous permet de définir un projet HRE existant sur un serveur distant, ou de modifier une telle définition, selon l'option sélectionnée dans l'affichage initial.

La sélection de « Modifier un projet distant » vous permet de choisir un projet distant dans la liste de tous les projets actuellement connus de votre installation HRE.
Une fois un projet distant sélectionné, l'écran s'agrandit pour afficher deux sections d'informations de projet modifiables.
Le choix de « Définir un nouveau projet distant » affiche les mêmes deux sections d'informations de projet, dans ce cas pour la saisie initiale.

Le premier ensemble d'informations correspond à la définition du projet distant : son nom, son nom de fichier et son emplacement sur le serveur distant.
Le deuxième ensemble d'informations définit le projet distant. Serveur - son nom, son adresse IP et son numéro de port d'accès.

Ces informations vous seront fournies par l'administrateur du serveur distant qui aura créé le projet HRE auquel vous souhaitez accéder.

Une fois les informations saisies, vous pouvez les enregistrer. Vous pourrez ensuite accéder au projet HRE distant défini via l'écran « Ouvrir un projet » de HRE.
