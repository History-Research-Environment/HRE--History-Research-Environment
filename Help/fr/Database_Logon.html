Connexion à la base de données

Cette boîte de dialogue apparaît lorsque vous tentez d'ouvrir un projet HRE défini sur un serveur distant.

L'écran affiche 3 zones de saisie de données, comme suit :
1. Données de connexion à distance, constituées du type Base de données ; Adresse IP du serveur; numéro de port du serveur. Ces informations sont principalement destinées à des fins de confirmation et vous n'êtes normalement pas censé les modifier.

2. Votre identifiant de connexion pour ce projet, tel que fourni par l'administrateur système du serveur distant et enregistré par HRE

3. Votre mot de passe pour ce projet, tel que initialement fourni par l'administrateur système du serveur distant

NOTE:

* HRE n'autorisera que 3 tentatives de connexion avant d'échouer le processus de connexion.

* L'utilisation d'un serveur distant nécessite que HRE soit en « mode serveur » - ceci est défini dans Outils->Paramètres->Mode serveur.
