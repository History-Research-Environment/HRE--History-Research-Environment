Gérer les types de sources

L'écran Gérer les types de sources vous permet de consulter, modifier, copier, ajouter et supprimer des données de type de source.

Lorsqu'un type de source est sélectionné, les modèles de ses structures de notes de bas de page et de bibliographie s'affichent également.
Pour modifier ces modèles, utilisez le bouton « Modifier » (ou double-cliquez sur l'entrée sélectionnée).

L'utilisation des fonctions « Ajouter » et « Modifier » affiche l'écran de modification du type de source, qui permet de modifier les modèles et le texte de rappel.

Un type de source en cours d'utilisation ne peut pas être supprimé.

Vous pouvez rechercher des types de sources dans la liste en saisissant du texte dans le champ « Rechercher ».
