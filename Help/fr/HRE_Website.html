Le site web HRE

La sélection de l'option site Web HRE ouvre votre navigateur par défaut et affiche la page d'entrée du site Web HRE.

Ce site Web contient le dernier bulletin d'information sur HRE (et leurs problèmes historiques), permet aux volontaires de s'inscrire pour aider au développement de HRE et contient d'autres éléments d'intérêt pour les utilisateurs de HRE.
