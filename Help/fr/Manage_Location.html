Gérer l'emplacement

L'écran Gérer l'emplacement vous permet de modifier les différents éléments qui définissent les données d'un emplacement, selon le style d'emplacement que vous avez sélectionné.

Vous pouvez également définir une plage de dates pour indiquer la période pendant laquelle cet emplacement était connu sous ce nom.

La section « Médias » permet d'ajouter des médias pertinents pour décrire cet emplacement.
