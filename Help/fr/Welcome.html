Que veux-tu faire ensuite ?

À ce stade, vous souhaiterez peut-être effectuer un certain nombre de tâches. Une liste de ceux-ci et comment les exécuter est couverte dans les paragraphes suivants.

Configurez HRE pour votre usage: nous vous suggérons fortement d'aller d'abord dans 'Outils -> Paramètres -> L'usager' pour personnaliser les paramètres HRE, en particulier vos paramètres d'emplacement des fichiers.

Pour ouvrir le projet Sample et l'explorer: dans le menu Projet, sélectionnez 'Ouvrir un projet', puis sélectionnez l'exemple. Cliquez sur 'Ouvrir' ou double-cliquez simplement sur l'entré Sample.

Pour ouvrir l'un de vos projets: s'il se trouve dans la liste 'Projets récents' du menu Projet, cliquez simplement dessus. Sinon, sélectionnez 'Ouvrir un projet', puis sélectionnez votre projet. Cliquez sur 'Ouvrir' ou double-cliquez simplement sur l'entré de projet souhaité.

Pour créer un nouveau projet vide pour vos données: dans le menu Projet, sélectionnez 'Créer un nouveau projet' et suivez les invites pour définir son nom et son emplacement.

Pour rechercher et ouvrir un projet qui vous a été donné par quelqu'un d'autre: dans le menu Projet, sélectionnez 'Ouvrir le projet', puis cliquez sur 'Parcourir' pour trouver le fichier qui vous a été fourni. Cliquez ensuite sur 'Ouvrir'.

Pour restaurer un projet précédemment sauvegardé dans le menu Projet, sélectionnez 'Restaurer le projet' et suivez les invites.
