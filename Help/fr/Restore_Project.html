Restaurer un projet HRE sauvegardé

Cet écran vous permet de restaurer les sauvegardes précédentes, puis de travailler avec ce fichier en tant que project ouvert.
Le projet d'origine, s'il est toujours présent, ne sera superposé si l'utilisateur l'autorise spécifiquement.

1. Cliquez sur TROUVER UN FICHIER DE SAUVEGARDE et dans le sélecteur de fichiers, sélectionnez le fichier que vous souhaitez restaurer; Sinon, accédez à n'importe quel autre emplacement et sélectionnez le fichier.

2. Votre sélecton, y compris son nom descriptif, sera transférée à nouveau dans la fenêtre de restauration du projet. Vérifiez les REMARQUES pour les options d'ici.

3. Définissez l'emplacement du dossier dans lequel la sauvegarde doit être restaurée, encore de la fenêtre Sélecteur de fichiers.

4. Le nom du fichier est par défault est celui utilisé par le projet d'origine, mais DOIT être modifié si la même destination doit être utilisée afin de ne pas superposer le fichier d'origine.

5. Il est essentiel de donner au projet restauré un nouveau nom descriptif pour le différencier du projet d'origine. Une fois que cela a été entré, cliquez sur RESTAURER.

6. Une fois terminé les projet restaurer sera le projet actif

7. Lors de la vérification, si le projet restauré est ce qui était requis, vous pouvez ensuite choisir de supprimer l'ancienne version du projet si elle est toujours sur cet ordinateur.

8. La deuxième partie de l'écran permet la restauration de fichiers externes (images, documents, etc.) à partir de leur fichier zip de sauvegarde crées dans Sauvegarde de projet. Notez que les deux parties de l'écran fonctionnent indépendamment - en d'autres termes, vous pouvez restaurer un fichier externe sans restaurer premièrement un fichier de projet.
