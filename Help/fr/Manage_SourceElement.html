Gérer les éléments sources

L'écran Gérer les éléments sources vous permet de manipuler les éléments sources dans toutes les langues prises en charge.
Par défaut, il affiche les éléments de la langue de données HRE actuellement définie.

Un double-clic sur un élément sélectionné vous permet de modifier son nom.

Le bouton « Ajouter » permet de créer de nouveaux éléments sources dans la langue choisie.

Notez que si vous ajoutez un élément dans la langue choisie, il n'y aura pas d'élément correspondant dans les autres langues.
Vous pouvez corriger cela à l'aide du bouton « Traduire », qui vous permet d'ajouter un élément équivalent dans une autre langue.

Vous pouvez rechercher des éléments dans la liste des éléments en saisissant du texte dans le champ « Rechercher ».
