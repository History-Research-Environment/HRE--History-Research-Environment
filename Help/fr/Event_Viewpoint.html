POINT DE VUE DE L'ÉVÉNEMENT

Le point de vue d'événement permet à l'utilisateur de voir toutes les informations enregistrées dans HRE pour l'événement sélectionné. Jusqu'à cinq (5) points de vue d'événements peuvent être ouverts pour chaque projet HRE ouvert.
L'écran s'ouvre initialement dans un format réduit affichant uniquement le nom, le lieu et la date de l'événement ainsi que 4 barres de titre de sous-écran.

Les 4 sous-écrans définis par les barres de titre se trouvent à gauche et à droite d'une barre de séparation verticale. Le séparateur peut être déplacé vers la gauche ou la droite pour modifier la quantité d'espace d'écran alloué à chaque moitié de l'écran. En haut du séparateur se trouvent deux petits boutons fléchés qui peuvent être utilisés pour que chaque côté occupe tout l'écran.

Chacun des sous-écrans peut être affiché en cliquant sur le triangle rouge à gauche de chaque barre de titre, ce qui change le triangle en vert au format pointant vers le bas et ouvre le sous-écran. Cliquer sur un triangle vert réduit le sous-écran. Chaque barre de titre indique également le nombre d'entrées contenues dans le sous-écran, par exemple « Images (2) ».

Lorsqu'un sous-écran est affiché, il peut également afficher les boutons "+" et "-" en haut à droite. Ceux-ci permettent d'augmenter ou de réduire le nombre de lignes de données affichées dans le sous-écran.

Si la hauteur de l’écran Viewpoint devient plus grande que l’écran HRE qui l’entoure, l’écran Viewpoint passera à un format déroulant.

Sous-écran Associés :
Cela affiche une liste de toutes les personnes associées à l'événement sélectionné.

Sous-écran Images :
Affiche toutes les images (le cas échéant) de l'événement.

Sous-écran Indicateurs :
Affiche tous les indicateurs actifs pour l'événement et leurs paramètres actuels.

Sous-écran Bloc-notes :
Les blocs-notes ne sont pas encore implémentés dans HRE.
