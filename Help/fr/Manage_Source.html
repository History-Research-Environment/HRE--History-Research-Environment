Gérer les sources

L'écran « Gérer les sources » affiche la liste des sources définies et le nombre de fois où elles ont été citées.
Cet écran vous permet de consulter, modifier, copier, ajouter et supprimer les données d'une source. Cependant, une source en cours d'utilisation ne peut pas être supprimée.

Lorsqu'une source est sélectionnée, les modèles de ses notes de bas de page et de sa bibliographie s'affichent également.
Pour modifier ces modèles, utilisez le bouton « Modifier » (ou double-cliquez sur l'entrée sélectionnée).

Le bouton « Ajouter » permet de créer une nouvelle source et le bouton « Modifier » permet de modifier la définition d'une source existante.

Le bouton « Afficher les sources inactives » affiche uniquement les sources marquées comme inactives, mais qui peuvent encore être présentes dans la base de données.

Les sources peuvent être recherchées dans la liste par Saisir du texte dans la zone de texte « Rechercher »
