Paramètres HRE

Cet écran vous permet de définir un certain nombre de préférences pour le fonctionnement de HRE. Les paramètres peuvent être modifiés à tout moment et ne sont pas affctées par la fermeture et la réouverture de HRE ou de vos projets.

1. Dans le menu principal OUTILS, sélectionnez PARAMÈTRES, puis UTILISATEUR pour sélectionner et définir une gamme d'options utilisateur

2. À partir de l’ensemble des onglets, vous pouvez choisir de modifier:

    HRE Look and Feel: Pour définir le "look et la sensation" - la taille et le type de police (la taille de la police (la valeur par défaut de la police est en Graphite, Blue-medium-font, Arial)

    Paramètres de langue: Pour définir les langues par défaut pour les écrans, données et rapports HRE

    Paramètres de format: Pour définir les formats de date et d'heure

    Options utilisateur:

  • Pour afficher l'écran d'accueil au démarrage de HRE (par défaut : Non)
  • Pour afficher les demandes de confirmation pour l'annulation et d'autres actions (par défaut : Oui)
  • Pour ouvrir le dernier projet HRE fermé au démarrage de HRE (par défaut : Non)
  • Pour sauvegarder un projet à sa fermeture (par défaut : Non)
  • Pour activer le traitement du code du plugin (par défaut : Non)
  • Pour recharger systématiquement le sélecteur de personne lorsqu'une personne est ajoutée à la base de données (par défaut : Oui). Si cette option est désactivée, le sélecteur de personne n'est rechargé que lorsque vous le demandez depuis le menu HRE
  • Pour toujours demander un nom de personne mariée lors de l'enregistrement des mariages (par défaut : Non)

    Accents: Pour définir les couleurs de la police et de l'arrière-plan pour les thèmes HRE que vous souhaitez définir

    Emplacements de fichiers: Pour définir les emplacements par défaut des projets, des rapports, des sauvegardes et des sauvegardes de fichier externes

    Options de débogage: Pour définir l'écriture d'un fichier journal (par défaut), la rédaction de messages de débogage et des messages de synchronisation. Ces options sont principalement destinées à être utilisées par un support technique HRE

    Mode serveur: Pour activer l'exécution de HRE en mode Server (par défaut est Non)
