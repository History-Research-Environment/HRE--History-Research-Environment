Quitter HRE

Cette option vous permet de fermer le projet ou les projets ouverts et de quitter le programme HRE, avec une option pour effectuer d'autres procédures avant la fermeture.

Il y deux façons de quitter HRE, et il y a de légères variations, mais le résultat final est le même.

A] DU MENU DU PROJET

1. Sélectionnez Fermer projet dans le menu principal du projet

2. Cliquez sur le project que vous voulez fermer pour qu'il soit mise en surbrillance

3. Vous avez maintenant cinq options :
- Afficher un résumé du projet sélectionné.
- Valider le projet, en s'assurant que les index et autres parties sont corrects et complets, et en corrigeant automatiquement les erreurs si possible.
- Sauvegarder le projet avant de le fermer, en vous donnant la possibilité d'enregistrer votre sauvegarde dans un emplacement choisi.
- Annuler la fermeture.
- Réellement Fermer le projet.

4. Lorseque chaque projet est fermé, un message vous informera que cela s'est produit, sauf si les invites d'annulation ont été désactivées dans les paramètres HRE

5. HRE se ferme lorsque tous les projets ouverts ont été fermés

6. Notez que le dernier projet fermé sera le premier projet qui sera automatiquement ouvert lors de la prochaine utilisation de HRE, si le paramètre HRE << Ouvrir le dernier projet fermé >> a été activé.


B] À PARTIR DU X EN HAUT À DROITE DE L'ÉCRAN PRINCIPAL HRE

1. Il vous sera demandé de confirmer que vous voulez quitter HRE, sauf si les invites d'annulation ont été désactivées dans les paramètres HRE

2. S'il n'y a qu'un seul projet ouvert, il sera fermé

3. Si plusieurs projets sont ouverts, l'écran Fermer project décrit ci-dessus s'ouvrira

4. Lorsque chaque projet sera fermé, un message vous informera que cela s'est produit, sauf si les invites d'annulation ont été désactivées dans les paramètres de HRE

5. HRE se ferme lorsque tous les projets ouverts ont été fermés

6. Notez que le dernier projet fermé sera le premier projet qui sera automatiquement ouvert lors de la prochaine utilisation de HRE, si le paramètre HRE << Ouvrir le dernier projet fermé >> a été activé.
