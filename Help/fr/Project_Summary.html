Résumé du projet HRE

Cet écran permet à l'utilisateur d'examiner un résumé d'un projet HRE en cours.

1. Le résumé du projet est un bref aperçu d'un projet HRE sélectionné.

2. Il peut être visualisé comme une option directe à partir du menu PROJET, ou peut être visualisé pour un projet ouvert choisi à partir de plusieurs des menus d'action: Ouvrir le projet, Sauvegarder le projet, Fermer le projet, Copier le projet (bref résumé), Renommer le projet, Suprimmer le projet et Quitter HRE (Fermer la procédure).

3. Le résumé montre:

  • l'heure actuelle
  • le nom du projet tel que sélectionné par le créateur
  • le nom du fichier utilisé pour accumuler les données du projet
  • le nom du dossier dans lequel il est stocké
  • le nom du serveur / ordinateur
  • quand le projet a été fermer pour la dernière fois
  • la dernière sauvegarde du projet
  • le type et la version de la base de données HRE
