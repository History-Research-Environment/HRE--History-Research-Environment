Modifier les types de sources

L'écran « Modifier les types de sources » vous permet de consulter et de modifier les données du type de source sélectionné.

Vous pouvez modifier le nom du type de source et éditer les trois modèles de notes de bas de page et d'entrées bibliographiques, ainsi que le texte de rappel.

Des messages d'erreur s'afficheront si vous tentez d'enregistrer des noms d'éléments de source inconnus ou d'utiliser des codes de formatage non valides, tels que [ITAL].
