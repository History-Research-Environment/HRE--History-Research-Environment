Administrer les styles de nom de personne et de lieu

L'écran Administrer les styles de noms de personnes permet à l'utilisateur d'accéder à une fonctionnalité très puissante de HRE: la possibilité de définir puis de mettre en œuvre une gamme de styles de noms et de styles de localisation pour s'adapter à une gamme de types et d'usages personnels, culturels et géographiques.

La disposition de l'écran pour l'administration des styles de noms (pour les noms de personnes et les emplacements) est illustrée dans le diagramme ci-dessous.


La section [A] contient des informations de base sur la disposition et l'utilisation de l'écran.

La section [B] contient une liste de tous les éléments actuellement définis pour être utilisés dans un style de nom, soit pour une personne, soit pour un emplacement, selon l'écran qui s'affiche. Comme le format et la fonction des deux types d'écran sont les mêmes, nous utiliserons les noms de personnes dans la description suivante, par souci de simplicité.

La section [C] contient une liste des noms de tous les styles actuellement définis du type choisi (personne ou emplacement) et lorsque l'écran s'ouvre, le premier élément de la liste est présélectionné. Une description plus longue du style de nom sélectionné est affichée dans la section [D] et peut être modifiée.

La section [E] contient une liste des éléments existant dans le style de nom sélectionné dans la section [C].

Les boutons pour Copier, Supprimer et Renommer les styles de nom dans la section [C] existent à sa gauche immédiate dans la zone (c). Certains de ces boutons peuvent être désactivés, selon le type de style de nom sélectionné - voir le tableau ci-dessous. La zone (c) contient également une case à cocher 'Par défaut' qui indique quel style de nom est le style par défaut (il doit y avoir un et un seul style de nom par défaut). Cette case peut être cochée pour changer le style par défaut.

Des boutons pour manipuler les éléments de nom dans la section [E] existent dans la zone (e). Celles-ci permettent aux éléments de la liste de la section [E] d'être réordonnés, supprimés de la liste, renommés et d'ajouter d'autres éléments contenus dans la section [B] à la liste de la section [E]. En outre, l'utilisateur peut ajouter de nouveaux éléments dans la liste de la section [B] à utiliser dans les styles de noms personnels. Certains de ces boutons peuvent également être inactifs selon le type de style de nom en cours de modification.

Le bouton 'Enregistrer' dans la zone (s) sous la section [E]) deviendra actif si les changements de style de nom doivent être enregistrés. Les modifications apportées à la section [C] sont enregistrées automatiquement et n'activent pas le bouton 'Enregistrer'.

La troisième colonne de l'écran, les sections [F], [G] et [H], contrôle les modifications apportées aux styles de sortie associés à tout style de nom sélectionné dans la section [C]. Les boutons des zones (f) et (h) fonctionnent de manière similaire, mais plus restreinte, aux boutons des zones (c) et (e). La zone (f) affiche également le type de style de sortie - soit pour les rapports ou l'affichage (à l'écran). Le type peut être changé, avec la restriction qu'un style de nom de personne doit toujours avoir un minimum de 2 types 'Affichage' (ou un pour les emplacements).

Le bouton 'Ajouter/supprimer une virgule' (z) sous la section [H] contrôle si l'élément sélectionné dans [H] a ou non une virgule après l'élément lorsque ce style de sortie est utilisé.

Le bouton 'Enregistrer' dans la zone (s) situées sous la section [H] deviendra actif si les modifications du style de sortie nécessitent une sauvegarde. Les modifications apportées à la section [F] sont enregistrées automatiquement et n'activent pas le bouton 'Enregistrer'.

Si le bouton 'Fermer' dans la ou la zone (s) situées sous la section [H] est cliqué alors qu'il y a des modifications non enregistrées, vous serez invité à enregistrer ces modifications ou à fermer l'écran sans enregistrer.

Règles contrôlant la manipulation des styles de nom

REMARQUE IMPORTANTE: lorsqu'un style de nom est copié, tous les styles de sortie associés sont également copiés. De même, si un style de nom est supprimé, tous les styles de sortie associés sont également supprimés.

Si des styles de nom ont été importés dans HRE à partir de TMG, des restrictions spécifiques existent sur les styles TMG, comme indiqué dans le tableau suivant. Étant donné que les styles de nom HRE sont beaucoup plus sophistiqués que les styles TMG, les utilisateurs sont encouragés à se convertir à l'utilisation des styles de nom HRE.
