Kopieer een HRE Project

In dit scherm kan u een HRE-project van elke bron naar elke gekozen bestemming kopiÃ«ren. Het kan dan naar behoefte worden geopend. De standaard kopieerlocatie kan worden ingesteld in Extra > Instellingen > Gebruiker > Bestandslocaties.

1. U kunt een project selecteren uit de lijst Bekende projecten of BLADEREN gebruiken om het venster Bestandskiezer te openen om een HRE-bestand op deze computer, op een externe bron of een verwisselbare schijf te vinden. Selecteer het gewenste project

2. Geef in KOPIEER NAAR het nieuwe project een naam

3. Gebruik opnieuw de Bestandskiezer om te BLADEREN naar de projectbestemmingsmap en wijs de nieuwe bestandsnaam toe

4. Druk op KOPIEER om het proces te laten plaatsvinden

5. Dit project kan nu worden geopend op deze computer of op een andere computer.
