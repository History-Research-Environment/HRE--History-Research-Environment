Sluit een Open HRE Project

Met deze optie kunt u een project sluiten dat momenteel is geopend in HRE.

1. Selecteer om een momenteel geopend project te sluiten PROJECT uit het menu en selecteer vervolgens de SLUIT PROJECT optie

2. De lijst met geopende projecten verschijnt, met projectnaam, bestandsnaam en of het een lokaal project is of een project op een server

3. Markeer de naam van het project dat u wilt sluiten. In de afbeelding is het project 'Mijn werkplaats' geselecteerd uit de momenteel geopende projecten, als er slechts één open project is, wordt dit standaard geselecteerd

4. Nadat u het gewenste project hebt gemarkeerd, verschijnt de naam hieronder om controle mogelijk te maken

5. Merk op dat op dit punt het geselecteerde project ook kan worden onderworpen aan andere processen, waaronder SAMENVATTING, VALIDEER of BACK-UP, en er is een mogelijkheid om het sluiten te ANNULEREN

6. Als je dan SLUIT PROJECT selecteert en de beslissing bevestigt, wordt het Project gesloten (er zal geen bevestigingsmelding zijn als die optie is geselecteerd in INSTELLINGEN). HRE blijft open zodat er desgewenst een ander Project geopend kan worden

7. Als het gesloten project momenteel het actieve project was, kunt u ervoor kiezen om een ander project actief te maken.
