DE NAAM VAN EEN PROJECT WIJZIGEN

Met deze menuoptie PROJECT kunt u de naam van een bestaand HRE-project wijzigen en de database op de bestaande locatie behouden.

1. Markeer het project waarvan u de naam wilt wijzigen. In dit voorbeeld heet het project "Immigranten". Alleen eerder geopende projecten kunnen worden hernoemd

2. Noteer dat u een momenteel geopend project niet kunt hernoemen (er verschijnt een pop-upadviesbericht) - het moet eerst worden gesloten

3. Ter controle kan een OVERZICHT van het gekozen project worden bekeken alvorens verder te gaan

4. Voer in de daarvoor bestemde ruimte de nieuwe naam voor dit project in. Spaties, cijfers, hoofdletters/kleine letters en gebruikelijke bestandsnaamtekens zijn toegestaan. De nieuwe naam moet een unieke projectnaam zijn, ongeacht de locatie ervan

5. Klik HERNOEM om het project een andere naam te geven of ANNULEER om niet verder te gaan

6. Het project is nu hernoemd naar "Nieuwe immigranten" in dit voorbeeld, en de lijst met projecten zal dan worden bijgewerkt

7. Het hernoemen van het project is alleen bedoeld voor gebruikerstoegang en identificatiedoeleinden, aangezien de bestandsnaam ("Immigranten_naar_AUS" in dit geval) ongewijzigd blijft op de computer.
