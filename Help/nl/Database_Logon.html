Databaseaanmelding

Dit dialoogvenster verschijnt wanneer u probeert een HRE-project te openen dat op een externe server is gedefinieerd.

Het scherm geeft als volgt 3 gegevensinvoergebieden weer:
1. Gegevens over externe verbindingen, bestaande uit het type Database; server IP adres; poortnummer van de server. Deze informatie is voornamelijk bedoeld ter bevestiging en er wordt normaal gesproken niet van u verwacht dat u deze wijzigt

2. Uw aanmeldings-ID voor dit project, zoals verstrekt door de systeembeheerder van de externe server en opgeslagen door HRE

3. Uw wachtwoord voor dit project, zoals aanvankelijk verstrekt door de systeembeheerder van de externe server

NOTEER:

* HRE staat slechts drie aanmeldingspogingen toe voordat het aanmeldingsproces mislukt.

* Voor het gebruik van een externe server moet HRE in 'servermodus' staan ​​- dit wordt ingesteld in Extra->Instellingen->Servermodus.
