Samenvatting HRE-project

Op dit scherm kan de gebruiker een samenvatting van een lopend HRE-project bekijken.

1. De projectsamenvatting is een kort overzicht van een geselecteerd HRE-project

2. Het kan worden bekeken als een directe optie vanuit het PROJECT-menu, of kan voor een gekozen open project worden bekeken vanuit verschillende actiemenu's: Project openen, Project back-uppen, Project sluiten, Project kopiÃ«ren (korte samenvatting), Project hernoemen, Project verwijderen en sluit HRE af (Sluitingsprocedure)

3. De samenvatting toont:

  • de huidige tijd
  • de projectnaam zoals geselecteerd door de maker
  • de bestandsnaam die wordt gebruikt voor het opslaan van de projectgegevens
  • de naam van de map waarin het is opgeslagen
  • de server-/computernaam
  • wanneer het project voor het laatst werd afgesloten
  • wwanneer er voor het laatst een back-up van het project is gemaakt
  • het HRE-databasetype en de versie.

Gerelateerde onderwerpen

  • Open project
  • Back-up project
  • Sluit project
  • Kopieer een bestaand HRE-project
  • Hernoem project
  • Wis project
  • Sluit HRE af en sluit open projecten
