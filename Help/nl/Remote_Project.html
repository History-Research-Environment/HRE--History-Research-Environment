PROJECTDEFINITIE OP AFSTAND

In het dialoogvenster Externe projectdefinitie kunt u een HRE-project definiÃ«ren dat op een externe server bestaat, of een dergelijke definitie bewerken, gebaseerd op uw selectie van een van deze twee functies in het beginscherm.

Door 'Een extern project bewerken' te selecteren, kunt u een extern project kiezen uit de lijst met alle projecten die momenteel bekend zijn bij uw HRE-installatie.
Zodra een project op afstand is geselecteerd, wordt het scherm groter en worden twee secties met projectinformatie weergegeven die u kunt bewerken.
Als u 'Definieer een nieuw extern project' kiest, worden dezelfde twee secties met projectinformatie weergegeven, in dit geval voor de eerste invoer.

De eerste set informatie is de definitie van het externe project: de naam, bestandsnaam en locatie op de externe server.
De tweede set informatie definieert de externe server: de naam, het IP-adres en het poortnummer voor toegang.

Al deze informatie wordt aan u verstrekt door de beheerder van de externe server, die het HRE-project heeft aangemaakt waartoe u toegang wilt krijgen.

Wanneer de informatie compleet is, kunt u deze opslaan. Toegang tot het gedefinieerde HRE-project op afstand is dan mogelijk via het normale HRE 'Open Project'-scherm.
