Externe projectdefinitie

In het dialoogvenster Externe projectdefinitie kunt u een HRE-project definiëren dat zich op een externe server bevindt, of een dergelijke definitie bewerken, afhankelijk van uw selectie van een van deze functies in het beginscherm.

Door 'Een extern project bewerken' te selecteren, kunt u een extern project kiezen uit de lijst met alle dergelijke projecten die momenteel bekend zijn bij uw HRE-installatie.
Nadat een extern project is geselecteerd, wordt het scherm uitgebreid met twee secties projectinformatie die u kunt bewerken.
Door 'Een nieuw extern project definiëren' te kiezen, worden dezelfde twee secties projectinformatie weergegeven, in dit geval voor de eerste invoer.

De eerste set informatie is de definitie van het externe project - de naam, bestandsnaam en locatie op de externe server.
De tweede set informatie definieert het externe project Server - de naam, het IP-adres en het poortnummer voor toegang.

Al deze informatie wordt u verstrekt door de beheerder van de externe server, die het HRE-project heeft aangemaakt waartoe u toegang wilt krijgen.

Zodra de informatie compleet is, kunt u deze opslaan. U kunt vervolgens toegang krijgen tot het gedefinieerde externe HRE-project via het normale HRE-scherm 'Project openen'.
